<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在一些“楼市玄幻小说”中，作者神秘兮兮地向你透露“发财秘籍”。当年零首付在广东深圳买多少套房子，升了30%猛赚一笔狠狠抛掉。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          其实这些都是很Low的外围小伙穷鬼A8编的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “零首付”毫不出奇。只要你下得了这个胆。遍地都是零首付。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          为此，甚至还出现了一个专门的炒楼流派：“零首付流”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          别的流派找房子，都是看房子价格本身的笋盘与否。或者是看某一地段有没有基建利好。或者是看某一类房产租售比安全。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而“零首付”流派与众不同。他们买房子，有且仅有唯一关注，“能否做低首付”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为各个楼盘之间差异很大。
         </span>
<span style="font-size:19px;line-height:150%;font-family:楷体;color:red">
          主要是银行的愚蠢
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          。一些很好的楼盘，银行并估不足价格。而垃圾CEO，银行往往会估得很高。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          零首付流”关键是深刻地了解
          <span style="text-decoration:underline;">
           银行内部
          </span>
          运作。他们知道哪些盘可以“溢高”。银行愚蠢地给出远高于市场价的估价。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          零首付流
          <span style="text-decoration:underline;">
           以银行为导向
          </span>
          。专门去购买“银行喜闻乐见”的房产。追求的极致是“
          <span style="color:red">
           首付尽可能少
          </span>
          ”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          零首付流也是一个特殊的流派。他们在房价上赚得并不多。有时候还会套牢，买的全是CEO盘涨得慢。而且高位接盘，手续费昂贵，智者不为。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但你仔细算算“自有资金收益”。零首付流绝对是可以排进前几位的。做得好的时候，甚至是第一位。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          2008
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          年的时候，外资银行刚刚进入中国。纸老虎的面目还没有揭穿。当时哥哥也是脑子发昏，找过去到花旗银行，问问能不能办二手房贷款。或者有吸引力的贷款产品。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          CITIBANK
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          的理财经理倨傲地拿出一张表格，上面是20个楼盘。并且说：“花旗银行对于房地产资质有着严格的要求，只有符合花旗标准的房产我们才接受按揭”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我接过一看，倒吸一口凉气。上面依次是：“
          <span style="color:#7030A0">
           汤臣一品，鹏利海景，财富海景，盛大金磐，翠湖天地，华府天地，锦麟天地，檀宫别墅………
          </span>
          ”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我说，朋友，你知道么，这些都是“落地打七折”的楼盘。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          2008
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          年翠二开盘八万，位置还不如隔壁的众鑫城。有一手没二手，你银行接这样的单子，才是真正
          <span style="text-decoration:underline;">
           银主盘
          </span>
          滞销资产，涨得不快走得不动。一旦砸在手里，根本处理不出去。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          花旗银行专员鼻孔高高翘在天空里。哥哥默默收下这张单子。翘起中指，暗地说了一声“SB”。外资银行刚刚进入中国，便已经流露
          <span style="text-decoration:underline;">
           其颟邗和官僚主义
          </span>
          。未来结局也就这样了，历史往往可以从青萍之末分析。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          言归正传，“零首付”并不是什么高科技。普通的马路中介，也懂得帮你操办写高一成。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而专门的“零首付流”，追求的是写高40
         </span>
<span style="font-size:16px;line-height:150%">
          ~
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          50%
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          ，这个就没那么容易。得专门的技巧和信息渠道，一般房子银行评估不接受。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而老派木乃伊多军，活该入土，很少做“高评”。主要是成本无法接受。换言之，借六还七，你觉得划算么？
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          三)
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          真实利率
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          借六还七，你的
         </span>
<span style="font-size:19px;line-height:150%;font-family:楷体;color:blue">
          真实利率
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          是多少。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这句话，首先要从“银行贷款”说起。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          假设银行有二种贷款；
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          1）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          十年期按揭贷款
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          2）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          五年期中小企业贷款，每次还息不还本
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          请问你选择哪一类。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          对于很多资金饥渴症的死多头来说，看见“按揭”二个字就眼睛发光。好比饿狼见到培根，恨不得狠狠扑上去。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但其实不对的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为“按揭”的本意。
          <span style="color:red">
           他和“信用卡分期”是一样的。前高后低。
          </span>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在《信用卡之分期》一文中，我们说道，12000元的分期，前高后低，本金不断归还。其实你只借了一半。平均只借了6000。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:blue">
          但是房贷，难道不是一样的道理么？
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          房贷同样是前高后低，借款本金不断减少。虽然房贷是一条非平均曲线，但是也大致匀直的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          所以房贷其实你只借了一半。
          <span style="color:red">
           说是30年100W的贷款，其实你只借了15年。
          </span>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因此在表格中，10年期按揭贷款，他和5年企业贷款是等价的。都是“只借了”五年。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          说回“借六还七”的话题。借六还七，相当于你增加了16.6%的“贷款成本”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而这一笔贷款成本，是需要你在未来的岁月
          <span style="color:blue">
           摊销
          </span>
          的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为你只借了十五年，所以平均下来，每年的“实际利率”上升1.1%左右。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          4.9%+1.1
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          ，相当于加了几次息房贷利率6%
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:12px;line-height:150%;font-family:楷体">
          （更精确的计算得用IRR）
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因此，虽然 “借六还七” 看似和古代农民高利贷“九出十三归”一样声名狼藉。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但因为
          <span style="color:#7030A0">
           摊销时间长
          </span>
          ，真实利率成本并不高。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但是，另一个方面，我们还需要考虑到，“30年期”往往是一种理想情况。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在绝大多数状态下，你是长持不到30年的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为对于“西点联盟”这样的进取新晋而言，
          <span style="color:#7030A0">
           他们最多持有2
          </span>
</span>
<span style="font-size:16px;line-height:150%;color:#7030A0">
          ~
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体;color:#7030A0">
          3
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体;color:#7030A0">
          年，就会将房产卖掉。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:#7030A0">
          最极端的情况下，不到1
         </span>
<span style="font-size:16px;line-height:150%;color:#7030A0">
          ~
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体;color:#7030A0">
          2
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体;color:#7030A0">
          年就会将房产卖掉
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这个时候，由于你写高了400W房价，并因此产生的40W元税费。就不能摊销到复利岁月中。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而成为了实实在在的“成本”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果你16.6%仅仅能用一至二年摊销的话，则每年融资成本高达8%。算上原有的4.9%，4.9+8=13%
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          Still
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          ，13%仍然是一个“可以接受”的范围。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          或者说，疯魔流通过“高评”手法，能以更小的资金获得更快的发展。他们是对的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          老一代多军手法保守，瞻前顾后，才是错误的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          四)
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          低评
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          和“高评”相对应的另一个手法，则是“低评”。反过来也是成立的。
         </span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            写高100W，增加贷款70W，付税费10W，净到手60W。借六还七。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            写低100W，减少贷款70W，省税费10W。净到手60W。还六得七。
           </span>
</p>
</li>
</ul>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “还六得七”。我只要拿85W现金出来，就可以净获得100W。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这相当于一笔“20%回报”的投资项目，尽管是“
         </span>
<span style="font-size:21px;line-height:150%;font-family:楷体;color:red">
          一次性
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          ”
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但是对于普通人来说，你上哪找20%的回报去。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          很多普通老百姓“不贷款，少贷款”的做法是错误的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你正确的做法，应该是
          <span style="color:fuchsia">
           尽可能地压低合同价。
          </span>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:fuchsia">
          把税省到最低。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:fuchsia">
          然后在力所未逮的范围内，贷款，贷满七成。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          以前有一个非常NB的网站，叫做“万网”（10000f.cn）,在这个网上，可以查到fangdi.com报给政府的过户备案价。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          每一个小区，哪一天成交哪一个房号。都可以看到。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这个网站非常之私密，仅限于内部少数人士知道。（已经关了）
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你如果登录万网，你就会看到，“写高”的人非常之少，“写低”是绝对绝对主流额。大约是99%：1%的样子。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          以
          <span style="text-decoration:underline;">
           2015年初永业公寓
          </span>
          为例，大约80%的成交集中在32000
         </span>
<span style="font-size:16px;line-height:150%">
          ~
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          35000
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          元/m左右。最极端的甚至可以做到19000/m
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而超过40000/m备案的一共只有一套。单价是65000.
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我们知道，2015年初的时候，永业价格早就达到了6
         </span>
<span style="font-size:16px;line-height:150%">
          ~
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          8
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          万。低于这个价格的根本就买不到。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          今天2016.07，永业的挂牌价大约在11
         </span>
<span style="font-size:16px;line-height:150%">
          ~
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          13
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          万。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          显然，32000/m的价格根本不可能买永业。而95%的登记备案价在这个区域。可见问题之严重。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          他只有一套是真正报备65000/m的。逻辑之下，65000才是真实的价格。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这个故事告诉我们，“职业炒家”是非常非常稀少之小众人群。在全中国也不会超过数千人。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:#7030A0">
          社会的99%，是“写低”人群。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          五)
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          全款
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          低评”到了极限，就是“全款”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为你本身贷款就不多。政府明文规定，上限70%。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果我低评一半。1000W的房子写到500W送件，贷款350W。则我实际贷款率仅有35%
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          贷得这么低，我干嘛还要贷款呢。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果贷款额仅仅才房价的一成，二成，我干嘛还要贷款呢。平白多一重手续，浪费宝贵时间。而且浪费贷票。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这样的情况下，催生了一个新的流派。“全款流”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          全款流的意思，是拿了1500W以上的现金，到处砸人。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          它和“老破小”流派不同。老破小凤姐变冰冰，也是全款支付。因为4个月一轮周期，你根本来不及贷款。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而“全款流”的区别。他砸人的是1500W现金，而不是150W现金。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为对于真正“大型”的物业。到了一定总价之后，接盘的人并不多。流动性也不好。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “急卖”的售主，有很多是中小企业主。急等着资金救企业。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而这些往往又是“非普通住房”。税费非常之高。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在这样的情况下，就是“全款流”饕餮。拿大额的现金砸晕业主，并要求降价。一般而言，可以额外砍下100W元左右。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而你全款价格可以写低，可以省税费。这样可以再省50W元左右。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          某些情况下，还可以“解药单”。获得更额外收益。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而你全款买下以后，再慢慢地处理抵押贷款。产证在手，有得是时间腾挪盘旋。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          六)
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          评估，贷控，打滚
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          银行的评估业，界限在哪里。行为准则如何。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          写高之后，上家获得更多资金。如何进行“贷控”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          交易中心写低，潜规则如何。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
