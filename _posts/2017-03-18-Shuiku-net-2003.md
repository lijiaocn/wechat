<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          尤其是燕郊的小火车，每天9元轰轰烈烈的进城。再喜感不过了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">

</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          第二个利益相关者，是商住。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “北京限购”原本对商住是一个巨大的利好。直到我们看到这条新闻。
         </span>

<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          对于“全国性”的买家，他们是在全中国布局的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “北上深”，不仅仅一线城市可以买。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而且重庆，天津，南京，杭州等二线城市也可以买。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          之前2016.11.28上海“认房又认贷”，购买力被逼去了北京不少。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我们知道，昨天最大新闻是“北京”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但其实昨晚纷纷扰扰，新闻还要多得多。广州，石家庄，郑州同时限购。只不过在京城人眼中，都算乡下了吧。
         </span>

<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          要建仓就抓紧了。别等门关了，空流泪啊。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          四）北京楼市
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          最后，回到我们普遍关心，核心的一个问题：“
          <span style="color:red">
           北京楼市会不会跌，价格会怎么走
          </span>
          ”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          每一次类似调控出台，看空群众欢欣鼓舞。此类问题恨不得问个二百次。最好能听到一声爹。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          有这个闲工夫，您上分答来问哥哥呀。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="text-align:center;line-height:150%">
<span style="font-size:12px;line-height:150%;font-family:宋体">
          （付费问答，666/问题。房产类排中国第三）
         </span>
</p>
<p style="text-align: center;">
<img data-ratio="0.5103092783505154" data-s="300,640" data-src="" data-type="png" data-w="388" src="{{ '/img/Ok4hZ0tV6r5CU9F6fHOicH8K0zHFia8JibicolPlpAISpibdq9taIIGKmm3b1pvngGjMty2WoicgAFpTVaGiaUNlAEAPg.png' | prepend: site.img | replace: '//','/' }}"/>

<p style="line-height:150%">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你要知道一件事。“限购限贷”这种事，不增加一平米的楼市供应。其本质是“
          <span style="color:red">
           交易成本
          </span>
          ”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          天底下的事，只要你不增加供应。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          长线来看，都是耍流氓。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          若要我打比喻的话，“认房又认贷”象一块刹车片。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          要知道，2017.1
         </span>
<span style="font-size:16px;line-height:150%">
          ~
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          2月，北京楼市的涨幅是非常迅猛的。一般认为每周3%。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “调控之后”，相当于踩刹车。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          原本迅猛狂飙的速度，有可能被刹下来了。按照惯性，或许还会滑行一段时间。预期可能还会有10%的惯性涨幅。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          然后就被完全刹车止住了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果我们相信各城市之间可以参照的话，2016.11.28上海“认房又认”调控之后，市场急剧降温。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          11月，12月，平均成交量仅有10000套左右。不足高峰期的三成。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:黑体">
          但是价格呢，纹丝未动
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="line-height: 150%; font-family: 楷体; color: blue; font-size: 20px;">
          你什么时候见过踩刹车踩到倒车的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          调控从来没使楼市下跌过，不要用股市的眼光来看待楼市。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为对于楼市，它有自己的特性。绝大多数的人，只有“一套多余房子”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我们知道，做买卖，术语称做“薄利多销”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-family: 楷体; font-size: 16px;">
          什么时候要降价促销呢。当我需要“跑量”的时候。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          好比我有1000套房子，有或者有1亿股股票。那我拿100套出来，“薄利多销”走走量掩护大部队。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但是对于楼市，尤其是二手房市场。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          绝大多数的卖家，都只有“一套多余房子”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          薄利多销，您开什么玩笑呢。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我一共就一套。还多销呢。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          当你只有“一套”的时候，你的游戏规则就变成了“吊起来卖”。慢慢卖，但一定要价格最大化。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果外部的市场环境不好。那怎么办。慢慢熬呗。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我反正也不急，我和你“耗长命”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          绝大多数的西城，东城房产，其负债率都是零。房东成本低到可以忽略。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我反正不急。您小孩要上学，你急。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这段“对耗期”一般会至少持续9
         </span>
<span style="font-size:16px;line-height:150%">
          ~
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          10个月。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          然后市场慢慢启动，慢慢恢复冰封期。真的要买房，总有办法的。再说新韭菜也在源源不断进入市场。
         </span>
</p>
<p style="line-height:150%">
<span style="line-height: 150%; font-family: 宋体; font-size: 12px;">
          (顺便说一句，交易量大幅下降。地方政府也坐不住，税少了)
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因此我们对“调控”政策的定位，就是刹车片。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          他可能使楼市减速，止步。一直到若干个月后重新启动。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但您见过刹车刹到倒车么，没有。
