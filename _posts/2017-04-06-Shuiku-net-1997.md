<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我不知道漳州的“人口死亡率”是多少。九个月够死多少人。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          1000个“交易对手”之中，有没有一二个突然挂墙的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但我们实战过的人都知道，在实战中，“药单”是层出不穷的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          正常交易，至少会有10%的单子，出意外岔子。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          专门的药单流就更不提了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果你真的去漳州。真的去买进500套，卖出500套房子。那我几乎一定肯定地可以和你说；
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          1）至少20套房子，会发生药单
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          2）至少30套房子，因为种种原因无法过户。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          3）至少40套房子，哪怕大涨之后，也无法沽清。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          真正等你把这“500套”全部卖完，会是一个漫长无比的过程。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你得专门留一个人在那里，专门处理遗留库存。还得好几年。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          譬如说，你淘宝店不想开了。想关门走人。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你剩下的“库存”要全部卖完，还得多少时间。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          哪怕你搞几轮“清仓大甩卖”，你试试看有多少效果。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这辈子我还从没见过没库存的实业。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          四）结语
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这种种事我们总结起来，答案就只有一句；
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:19px;line-height:150%;font-family:微软雅黑,sans-serif;color:blue">
          编，你使劲编
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:19px;line-height:150%;font-family:微软雅黑,sans-serif;color:blue">
<br/>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p>

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
<br/>
</p>
<p style="line-height:150%">
<span style="font-family: 楷体; font-size: 16px;">
          魏广华的故事，你买一套二套可以，买三套五套，也能碰碰运气。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          十套以上，这样的“
          <span style="color:red">
           玩法套路
          </span>
          ”肯定是玩不转的。肯定遭遇无数无数瓶颈，成本呈指数上升。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:微软雅黑,sans-serif;color:blue">
          无耻的文科生小编，无非是知道了一二例案例。然后把事实无限扩大，自动
         </span>
<span style="font-size:27px;line-height:150%;font-family:楷体;color:blue">
          Ｘ500
         </span>
<span style="font-size:16px;line-height:150%;font-family:微软雅黑,sans-serif;color:blue">
          上去。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而“造谣”的结果，对我们整个社会的危害是非常大的；
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          1）在民众中，传播了恐慌情绪
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          2）在民众中，传播了仇恨情绪

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          3）在民众中，传播了弱智故事

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          4）在民众中，传播了浮躁心态
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
