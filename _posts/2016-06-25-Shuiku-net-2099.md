<h1 style="line-height:150%">
<span style="font-family:宋体;color:#984806">
          借款流基础概念
         </span>
<span style="color:#984806">
<sub>
           #2150
          </sub>
</span>
</h1>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          成年人的道路，艰辛，险阻。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但回报也多十倍。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          一）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          撸分流的缺陷
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:12px;line-height:150%;font-family:楷体">
          （延续上篇）
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在中国90%左右的卡坛内，其主要的流派是“撸卡积分流”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          撸卡是一件非常令人兴奋的事。积分流大师整天在那里炫耀，又找到一个洞，又找到一个洞。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          以一张金卡5W为例，千二折算。50000积分=100元。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          也就是“撸积分”的人，每次可以赚上100元。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          赚钱这种事，“小额多次”是最最最令人幸福的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          好比同样给你十万，一次性给，高兴劲最多也就二三天。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但是隔三岔五的给你100，300，500，感觉就像是遇见财神了。能快乐一整年。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            想要一天就有回报的，你是快递外卖散工
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            想要一个月有回报的，你是工薪上班族
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            想要三五年获取回报的，你是职业经理人。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            想用一辈子等候回报的，你是公司老板。
           </span>
</p>
</li>
</ul>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          撸卡这种事，因为回报给得太多太散，每一次都是100元。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          积少成多，年终算一下总账。其实总收入并不多。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          就总体而言，“撸卡流”基本上一年约有四五万的年收入。而某些职业的，纯粹靠这个吃饭的，一年收入可以达到20万人民币。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          200000 RMB = 1
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          亿积分
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而再往上走，撸卡撸到一年百万（月入十万），这样的人全国肯定会有。但屈指可数。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          为什么，你细想下去，“撸卡流”同样有很多限制。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          1）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          不是每一个银行都可以撸的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          撸卡最主要的是工行，招商，光大，网付三大巨头。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          可是光大的积分基本是津币。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          ICBC
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在2013年之前只有积分功能，没有积分兑换功能。堪称神迹。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          CMB
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          基本就是一个骗屌丝银行。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而另一些“积分”很值钱的银行。例如民生，浦发，建行。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          对积分的管理要严格很多。尽管也有疏漏。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          2）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          不是每一个时间段都可以撸的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          对于撸卡流来说，他是一个不停地“发现新洞”和“塌洞”的过程。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你找到了一个撸分漏洞。但是银行不是傻瓜。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          最多过了三四个月，银行逐渐发现问题。就把洞堵上了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          所以这是一个“围剿反围剿”的过程。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你上BBS学学攻略，第一笔小钱赚得很开心。但是过三个月洞就塌了。如果你想要长期赚钱，你必须长期泡在里面。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          天天找，天天挖空心思钻新的漏洞。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “长期”的成本就很可怕了。相当于占去了你的本职工作。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          所以撸卡能撸到上千万积分的人也不多。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          3）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          银行的反击。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果你是民生银行的风控。全中国信用卡积分“上亿”的人有几个？
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          不客气地说，按照3000万张的发卡量，仅仅“卡内积分”超过50000分的，就可以筛到0.1%
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          可卡内积分超过500万分的，可能是0.001%
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          卡内积分超过1亿分的，这实在太耀眼了，躲也躲不过…………
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          银行又不是傻子。这事也是有人盯着看着你的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          别的不说，一页对账单35条。如果你的消费记录超过350条，你就要寄十封对账信。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          光这件事，就足够传得卡部人皆尽知了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          所以“撸分流”做不大。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          那群人兴高采烈地拿点银行礼品、积分，是可以的。玩大了，上面都是瓶颈。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          二）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          借款流
         </span>
</p>
<p style="line-height:150%">
<br/>
</p>
<p style="line-height:150%">
<img data-ratio="0.34172661870503596" data-s="300,640" data-src="" data-type="jpeg" data-w="" src="{{ '/img/Ok4hZ0tV6r55BIHo1RwIA6FzW57khicK7wy11naD0cvibV5tMlu4xW4hCfNEvRZCe3tkXOxxicn0Emfibicln9yJJyQ.jpeg' | prepend: site.img | replace: '//','/' }}"/>
<br/>
</p>
<p style="line-height:150%">
<span style="font-family: 楷体; line-height: 150%;">
          对于“透支借款流”来说。他们是非常非常看不起“积分流”的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          为什么，你凭什么看不起人家。有什么标准。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          标准只有一个：“赚钱的多少”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          对于积分流来说，他们大致是赚“几万
         </span>
<span style="font-size:16px;line-height:150%">
          ~
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          十几万”的数量级。每一年。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          对于“借款流”来说，基本是赚“几十万
         </span>
<span style="font-size:16px;line-height:150%">
          ~
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          几百万”的数量级。每一年。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          也就是相差十倍。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          或者你这样想。每一笔积分的利益，是“千分之二”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而我们做预算，基本上可以做到24%年回报，也就是每个月2%。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          坐着不动，相当于你蹦跶十次。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而24%这个数字怎么来的呢。咳咳，咳咳，见之前“
          <a data_ue_src="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=2651014724&amp;idx=1&amp;sn=6f8e68cd2c77c1e7fe2d4c321d714456&amp;scene=21#wechat_redirect" href="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=2651014724&amp;idx=1&amp;sn=6f8e68cd2c77c1e7fe2d4c321d714456&amp;scene=21#wechat_redirect" target="_blank">
           楼市的回报有多高
          </a>
          ”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这个24%还是说得低的。更成功的人士，大致是透支300W，年赚300W。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “借款流”和“积分流”是完全不同的科技树发展方向。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在于“借款流”看来。你辛辛苦苦地把钱借出来，T+1又急吼吼地还进去，是严重的“买椟还珠”行为。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “借款流”的另一个特色，他是要承担“风险，痛苦，压力”的。
         </span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            如果你每天都结算工资，你是送外卖的。
           </span>
</p>
</li>
<li>
