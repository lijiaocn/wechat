<p style="line-height:150%">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          一）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          保险是什么
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          保险最早是“古典保险”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          好比说，你一个40岁的中年人，办公室白领，生活在一个中国上海这么安全的城市。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          则你的死亡率，是非常非常低的。或许只有1/700的风险。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          现在保险公司卖你一份保险，他的赔率是1/550
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          保费1万，赔偿金550万。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果有700个人买这份产品的话，最终保险公司收入700W。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          赔付550W。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          留下150W。再付掉办公行政费用，销售费用，政府税收，赚40
         </span>
<span style="font-size:16px;line-height:150%">
          ~
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          50W
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          按照正宗保险公司的术语。他们称之为“费差损”“利差损”“险差损”。
         </span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            预估1/700，实际死亡率1/750，这部分叫险差损。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            预估行政费用80W，实际花掉70W。这叫费差损。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            预估投资回报率2.2%，一年后实际得到2.5%，称之为利差损。
           </span>
</p>
</li>
</ul>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          保险公司的利润，靠着“险差损”的套利空间。这就是“古典保险”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          二）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          现代保险
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          现代保险业的发展，在于保险公司觉得“赚得太少”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          他们不满足你投保“保费”10000元。他们希望你“投保100W元”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          那怎么做呢。保险公司其实是“捆绑销售”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          他卖你一份保单，价值100W元。其实里面的“保费”只有1W元，人生死亡赔付550倍。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          剩下的99W呢，他们去买了一个“债券基金”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          然后你这个基金会有各种“运作”，每个月都会给你寄财务报表。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          表面上你买保险“保值增值”会有回报，其实你就是买了一个股债基金。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果我们进一步深入地看下去呢。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          保险公司这个“基金投资”，他们并不是直接投资的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          保险公司往往也是到市场上，再去找华夏基金，易方达，博时之类的“二传手”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          既然这样，你为什么不直接买“基金”呢。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:19px;line-height:150%;font-family:黑体;color:red">
          平白多一重剥削。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          更严重的是，保险公司往往会收取非常高的“基金销售费率”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          每一张保单都有“现金价值”。如果你上网查的话，这个数字
          <span style="color:blue">
           大约在6%左右
          </span>
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你买100W的保险，基金公司只拿到94W元，作为初始投资。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          还有的6W，就直接被保险公司当作“销售精英”，归入利润栏去了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而且保险公司往往还有非常苛刻的条款。三十年内都不允许退保。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          一旦退保，有非常严厉的罚金。或者份额打折。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          更恶心的是，最近保险公司还纷纷“违约”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          说由于投资失败，投资回报率不足。以前发产品时的承诺无法达到。要求客户晚取款或者追加投钱。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          反正解释权都在细字合同里。真是无耻无下限。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因此，如果你身边哪一个朋友，以“连投性保险”作为投资工具。或者将大额资金投入保险。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          则这个人智商一定是有问题的。脑子不好使。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在整体金融圈内，“保险业”也是非常让人瞧不起的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在整个歧视链中，处于非常下游的地位。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          卖保险的，大致也就和卖传销的领带差不多。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:21px;line-height:150%;font-family:楷体;color:blue">
          这也是我们的“资产组合”中，不包含任何保险的原因。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          三）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          炒房业中的保险
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为我们需要一类特殊的保险，“人身死亡险”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          要知道，以家庭为单位的话，“炒房”这活，往往是夫妻中一个人干的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而炒房是一种“精力，体力”密集型。对个人的素质要求非常非常高，对勤奋严谨踏实的要求非常高。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你设想一下，假如你重病，突然植物人六个月。你LP能不能扛得住。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          当你仔细的，深入地思考一下。
         </span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            信用卡空当接龙

</p>

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            外部私人借贷
           </span>
</p>
</li>
</ul>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          一旦你出个三长二短，外部的私人借贷债主顿时坐不住了。好歹要上门探视一二。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这样的情况下，家庭主妇你LP能否撑得住，是一个大问题。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          所以我们有一个“缺口”。一个风险敞口缺口。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<span style="font-size:16px;line-height:150%;font-family:楷体">
          ：所有的信用卡负债
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          ：所有的私人借款负债
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          C
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          ：“月供-房租”*24
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          D
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          ：适当的医疗费

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          将A+B+C+D加在一起。就是你需要的“风险敞口”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          四）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          人身意外保险
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “人身意外保险”，或者称为“死亡保险”。是一种赔率最高的保险。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          以40岁中年白领为例。一般他的赔率，在1：550
         </span>
<span style="font-size:16px;line-height:150%">
          ~
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          570左右。
          <a name="_ftnref1" style="font-size: 10px; text-decoration: underline;" title="">
<span style="line-height: 150%; font-family: 楷体; font-size: 10px;">
            [1]
           </span>
</a>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果你每年投保1万元。则“意外身故”可以赔偿550W元。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我们常常在“柯南”侦探小说里，听到“投保大额保险，意外身故”。讲的往往就是这种高杠杆保险。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你要注意，这1W元是“净消费”的。也就是一年之后，什么也没有，10000元平白消失的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因此这种“纯风险”型保险，往往不被老派人士接受。不被父母辈们喜欢。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          他们喜欢的是；
         </span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            投保40W元，保额550W元。一年之后40W元原额退还。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            投保400W元，保额550W元。一年之后返410W元。
           </span>
</p>
</li>
</ul>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          至于老派人士的数学。我也就呵呵。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          划算不划算，你自己算算。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我们继续以1:550倍为例。投保1W元/年，保额550W。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你先算一算你的A+B+C+D，看看一共需要多少。一般情况下，对于小白，550W元足够了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          某些情况下，对于某些负债特别大，做人又特别抠，现金特别紧张的人，譬如叔叔。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你说我每年花1W保险，我还是有点心疼的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          则你可以投保5000元，保额275W。相当于A+B+C+D的
          <span style="color:blue">
           50%
          </span>
<span style="color:blue">
           覆盖率
          </span>
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你摊到每平米房贷上，也没多少钱。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          最后申明一下，如果你真的按本篇攻略去做了。真的投保5000元，要求银行给你开一张275W的保单。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你最有可能的结果是：开不出。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为对于中国国内的银行，目前基本都取消了“纯保障险”。也就是那种1:550的保险，他们不卖了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          保险公司黑透了心。他们希望卖你那种“40W保550W”或者“400W保550W”的产品。

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">

<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
