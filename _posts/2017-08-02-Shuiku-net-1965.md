</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          这背后其实是一种哲学思想，即“放弃”的力量。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          你愿意放弃某样东西，
          <span style="color:red">
           承受某种不可逆的永久性的损失么。
          </span>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          自古以来，凡是成大事，做伟大事业的人。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          他们所追求的目标，必然是天高深远，困难重重的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          “没有谁能随随便便成功”，因为目标太遥远，代价太高昂。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          你想要“无损”是不可能的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          我们看到目前社会上的年轻人，他们存在非常严重的“中二病”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          简言之，完美主义。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          换言之，公主病，玻璃心，草莓心，小皇帝，圣母婊，五毛党，国货喷子。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;color:red">
          心比天高，薄如瓷瓶。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<blockquote>
<p style="line-height:250%">
<span style="font-size:24px;line-height:250%;font-family:微软雅黑,sans-serif;color:blue">
           没有牺牲，没有残缺。没有永不可逆的损伤。
          </span>
</p>
<p style="line-height:250%">
<span style="font-size:24px;line-height:250%;font-family:微软雅黑,sans-serif;color:blue">
           注定你毫无力量。
          </span>
</p>
</blockquote>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          在约翰·特拉沃尔塔电影《豪斯医生》中，女主角为了陷害男配角，切掉了自己的子宫。从而导致终生不育。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          俾斯麦统一德国，中欧崛起。为了获得英国对于“强权崛起”的谅解，他彻底放弃海军。把海上力量，削减到几似于零的屈褥位置。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          可是到了“愤青”威廉二世的手中，为了口炮党们的欢呼，他花费了120个陆军师的预算，修建了全球第二大，而几乎没开火的“日德兰”海军。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          缺了这笔钱，陆军就能打赢一战。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          英王金雀花王朝为了赢得海外霸畎，不惜永久性地放弃祖先的领地“诺曼底”。和法国和解。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          二战后，西德为了和盟军和解，不惜签署“德法煤铁协议”。永久性地放弃战峥的实力。直到下一次工业革命。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          这些行为，在当时都被骂得狗血喷头的“卖蝈贼”。但其实却是老练的政治家，深谋远虑做出了长久手交易。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          前二天看美国电影《异星觉醒》，深深感觉美利坚快完蛋了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          当时，异形外星人已经爬出了培养皿，已经绞干了一个黑人的血。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          这个时候，USA宇航员们要做的事，不是赶紧封闭上舱门。禁闭得严严实实的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          甚至最好把飞行舱隔离，最好分开飘在太空。
         </span>
<span style="font-family: 楷体_GB2312; font-size: 16px;">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          相反的，“政治正确”二个男的发疯一样喊“快开门”。只要还有一口气，赶紧冲进去救人。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          外星生物因此连杀三人，逃出实验室。24Ｋ纯SB，猪队友。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          请再抬头看看这一行红字：
         </span>
<span style="font-size:24px;line-height:150%;font-family:微软雅黑,sans-serif;color:red">
          承受不可逆的永久性的损失。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          人命关天_
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;color:blue">
          美利坚已经完蛋了。他已经承受不起牺牲和放弃了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          不愿牺牲，你的选择就大大减少。dT&lt;0，世上又有几件完美无瑕的事。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          造铁路，修大坝，建电厂。猫猫狗狗的野生动物就死几个吧。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          六）结语
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          如果你学习经济学的目的，是“富国强民”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          是“统筹和调度”，把所有的资源，分配到最高效率的地方。是生产。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          则你的经济学，多半是学不好的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          哪怕学好了，也是小乘境界。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          经济学真正的精髓，是“放弃”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          所谓经济学，其实是“我愿意放弃什么，我想要得到什么”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          神魔皆以血饲。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          “凡人，你想要什么”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          “你愿意放弃什么”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
