</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你看怎么办，你这个万恶的资本家要不要拔一点血出来。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你要是不服气的话，要不要让IRS来查查你的合规性。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
<img data-ratio="0.9933333333333333" data-s="300,640" data-src="" data-type="jpeg" data-w="600" src="{{ '/img/Ok4hZ0tV6r7iatS0yan67KKoKsJhYz0icJNqrpEuSEkdn57CvtsuPU2iaXITx9SoJwmpDsrRiaCJ24TOndIDdtAC9Q.jpeg' | prepend: site.img | replace: '//','/' }}"/>
<br/>
</span>
</p>
<p style="line-height:150%">
<span style="font-family: 楷体; line-height: 150%;">
          你若要是企业总部，你就必然有很多机器设备。
         </span>
<br/>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          尤其是高端制造业，高端医疗器械，你可能还有极重要的“母机”。全世界就这么几台，达到了极高的精度。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这些机器搬迁不易，卡耐基也搬迁不易。留在市政府手里，就是最好的人质。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          所以美国的CEO们，是想得极为明白的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          惹不起，我还躲不起么。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我就去人口最少最少的行政区小镇。3000人还太多，恨不得300人。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          当镇长找上门来，要我支助年轻人学费。可以，一口答应。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          老年人养老金。可以，一口答应。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          医疗保险计划。可以，一口答应。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<blockquote>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
           你撑死了也就3000人。我就算把你这3000人全部都喂饱，又如何。
          </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
           但是我打死也不会去纽约。1400W人口的纽约新泽西，发起疯来；
          </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
           那可真是把我抽干吸干也填不满呀。
          </span>
</p>
</blockquote>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          七）IT和金融
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          言归正传，回到我们的标题《为什么是IT和金融》。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          其实你举一反三。读到这里，自然已经明白了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我们要感谢芬·布朗于1926年发明了火箭Rocket。否则这玩意放在今天，是无论如何不可能被发明的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          相应的，也就不会有气象卫星，铱星通讯，GPS指路。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在今天，“太空电梯”，“太空农场”，“磁悬浮城市”这些尝试，是永远永远也不可能发生了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你想，火箭发射时，喷出了多少火焰气体。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这些气体，会不会污染环境，会不会破坏臭氧层
          <a name="_ftnref3" style="font-size: 10px; text-decoration: underline;" title="">
<span style="line-height: 150%; font-family: 楷体; font-size: 10px;">
            [3]
           </span>
</a>
          。要不要搞几千个人在门口抗议。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          哪怕不污染环境，火焰喷死了某一个蝴蝶的亚种。火箭发射基地，导致了某一类鼹鼠亚亚亚亚种的灭绝，也是非常有可能的事。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          环保份子永远存在。很多时候，环保是一门“收保护费”的行为。真正肥润养活的，是环保组织领袖。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “保护费”很高，封口费很高。如民主社会的毒瘤。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          20世纪下半叶，任何大型的创新与探索行动。都不可能在欧美发生。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因此，我们可以回答标题《为什么是IT和金融》。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          为什么百业萧条，人类文明缓慢。而只有IT和金融，还在创新不断，活力不断，生机不断。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:24px;line-height:150%;font-family:微软雅黑,sans-serif;color:blue">
          因为这二个行业，环保分子管不到！
         </span>
</p>
