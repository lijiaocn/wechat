<h1 style="line-height:150%">
<span style="font-family:宋体;color:#984807">
          沉默的上海人
         </span>
<span style="color:#984807">
<sub>
           #400
          </sub>
</span>
</h1>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          直到今天，上海土著仍拥有着66%的人口，50%的收入，和75%的财富。
         </span>
</p>
<p style="line-height:150%">
<span style="font-family: 楷体; line-height: 150%;">
<br/>
</span>
</p>
<p style="line-height:150%">
<span style="font-family: 楷体; line-height: 150%;">
          但是在房地产的所有购买研究中，似乎已经把这个群体遗忘了。
         </span>
<br/>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          一）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          有多穷
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          首先，我们要说一些关于土著的事情。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          有一件事情许多人都没有搞清楚。在过去25年中（1990-2015），增长最迅速的群体是哪一个。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          许多人会回答，是“新移民”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这个回答也是显而易见的。900W新移民蜂拥而入，填满了浦东到闵行的每一寸郊区。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而且新移民作风泼辣，挥金如土，买起5W/平米的房子眼睛都不眨。代表了旺盛勃勃的生机。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但是这个回答，却是错误的。过去二十五年，实力增长最快，面貌改天换地，反而是很多人以为“垂垂老暮”的浦西土著。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          为什么，也没觉得浦西人现在多富呀。都没几个土豪。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这样说法，关键是你没意识到1990年之前上海人有多穷。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          和绝大多数人的印象不一样，1990年之前的上海，其实是一个“贫穷落后”的城市。而远远不是所谓“远东第一大都市”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          当时的上海，和现在上饶，临沂档次差不多。远远比周边的常熟，上虞要穷得多。每年有很多的上海女孩子嫁到江浙，因为浙江人更富，要抢着嫁。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          上海人谈论的，是去广州深圳打工，深深地艳羡那里二三倍的高工资。烦恼的，是要不要放弃国企铁饭碗身份。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          今天大家都在骂，闸北火车站导致了脏乱差，把整个地段价值全都扯低了。还造成了南北交通的大堵塞。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而事实的情况呢，周润发“上海滩”中的火车站，原本是“老北站”。是在七浦路上苏州河北。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          1987
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          年时，是闸北区政府哭着喊着，满地打滚哀求着市政府，把火车站搬到闸北去的，救救闸北。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          火车站搬闸北之后，立刻对当地的经济产生了立竿见影的拉动效果。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          闸北区居民，长期以来叫做“滚地龙”区民。滚地龙什么意思，就是裹张草席，往马路上一躺。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          意思就是“这户人家完全没有固定资产”。是彻彻底底的无产阶级。什么彩电冰箱，连个热水瓶都没有的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而火车站一搬闸北，整个闸北经济就活了。你躲在路边卖个凉皮，开个小旅馆，十几万人就有了收入，有了生计，告别无产阶级。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          说这个例子，告诉你的不是上海人有多富，而是当年有多穷。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          二）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          大赢家
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果我们以2000年为地产开发分界线的话，则2000年之前上海的住宅存量，在4.5亿平米左右。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          基本可以认为，这些房子全部在“土著”手里。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          时至2015年，这些房子，平均升值都在4W元以上。也就是以前3000元平米的房子，现在43000元/平米。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这是什么概念呢。4.5亿*40000元=18万亿。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这18万亿，是平均分配给1400W土著居民的。也就是
          <span style="color:#E46C0A">
           每人130万元
          </span>
          ！
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          三口之家，户均400万。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这个18万亿是什么概念呢。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          截至到2015年，上海市GDP（预）是25500亿元。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          一般情况下，国家税收要拿掉30%，企业收入35%，能分给居民发工资的不会高于35%。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          也就是工资8500亿左右。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这还是2015年的数字。前二年的GDP更低。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这说明什么呢，“人赚钱难，钱赚钱易”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在过去十五年内，哪怕你全部的工资累计，一分钱不动。总数可能8万亿都不到。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而上海居民的资产升值是18万亿！
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这同时也回答了一个问题，“土著和高薪大学生哪个更有实力”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在8500亿工资中，假设外来劳动人口占50%，一年工薪收入约4000亿。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你累计十年，也不过数万亿的规模。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而土著手里的5亿平米，每年涨2000元/平米，一年就是1万亿。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而上海楼市的涨幅，远远不止每年二千。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          土著和外地小白领的实力增长，大约是400
         </span>
<span style="font-size:16px;line-height:150%">
          ~
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          500%
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          ：1的样子。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          菜总说“二本外地狗，摁倒拆迁妹”，实在是宇宙不破的真理。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          三）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          幸运
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我们要心存感激，要铭记幸运。今天所有的幸福，不是靠你劳动所得，不是靠上海人的勤劳与智慧，而拥有如此财富。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我们是幸运的，当年中央选择了“上海开发”，而不是“无锡开发”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果当年开发的重点放在了无锡，无锡房价43000/m，而上海核心区的房价，则如蠡湖大街一样8000元/平米。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这里面就是18万亿的财富差异。
          <span style="color:blue">
           户均400万元
          </span>
          的财富差异。每个上海家庭少了四百万，立刻和瘪三差不多了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:red">
          没有了外地人涌入，没有城市开发，上海人就是滚地龙瘪三
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我们一定要用最大的热情，欢迎外地人涌入，支持外地人涌入，打造最热情好客的城市。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          今天我们所拥有的一切，全缘由“是上海，不是蠡湖”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          四）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          剥削
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          现在我们来说一说，“外地人”创造的财富，是怎样落到“土著”本地人手中的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你这400万元/户，是怎么来的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这主要通过“宏观剥削”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          香港人曾经有过一个笑话，玩具大王，电动马达大王，龙虾大王，无论你赚了多少钱，最终你都是为“房产商”打工。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          上海的逻辑，与此类似。他通过几个管道，吸干外地人的血：
         </span>
</p>
<ul class="list-paddingleft-2" style="list-style-type:num1;">
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            以极高的价格，将浦东土地售予外地小白领
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            将市中心豪宅卖给温州土豪
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            55
           </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
            平米限制，售出老公房
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            交易税费
           </span>
</p>
</li>
</ul>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          其中1，2我们之前已经有所阐述。而3会在下一篇讲。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          五）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          官僚
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            2015
           </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
            年房地产业销售6.4 万亿元;
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            缴纳契税2874亿元;
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            房产税1372亿元;
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            营业税4051亿元;
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            土地增值税2719亿元;
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            缴税合计约1.1万亿元;
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            银行房贷余额12万亿元;
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            获8400亿元利息;
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            土地收入28517亿元;
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            政府和银行从房地产获得收入47917亿元;
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            占6.4万亿元收入的75%。
           </span>
</p>
</li>
</ul>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在房地产开发中，一般认为75%的金额被政府拿走。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而政府是什么。政府“广而泛”的名词，归根到底政府还是“人”组成的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在
          <span style="font-size: 16px; line-height: 150%; font-family: 楷体; color: rgb(79, 97, 40);">
           生态位
          </span>
          中，外来的大学生白领精英（简称白骨精），虽然他们占据了工蚁和黄牛的位置；
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          可是就“政府”而言，几乎95%的职位，都是被本地人土著垄断的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          虽然我们时常谈论着上海“书记、市长”等顶级官吏，往往由中组部指定。外省的官员居多。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          可是你看到中层官僚，厅局处科四级，几乎全部都是本地的土著上海人。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          上海有多少名处长。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          正确的数字大概是50000名处长。其中每一个人都有40
         </span>
<span style="font-size:16px;line-height:150%">
          ~
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          50W
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          的年收入。不亚于一个金领白领。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          别看外地小白领自以为“高薪精英”，可是在“官场”这一个生态位的竞争之上，他们几乎全军覆没。浦西才占据着上海的权力中心。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          官员有什么用呢。政府拍了一块地，有上百亿的收入。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          可是这些收入，他还得花出去。简单点有中建一二三局，要兴建道路水电和高架。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          比较复杂点，还有绿化养护，交通火警。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          然后还有社会养老，慈善统筹，政府运作。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而这一些钱，几乎全部都落入了不显山不露水的“官场中人”手中去了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          最简单的一点，外环线绿化养护工程，你知道其利润和成本是多少么。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          上海人这些年是明显滋润起来了。不仅国企效益很好，而且公务员也不再是紧巴巴。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          各个国营系统，因为有政府支持，也不再是清水衙门。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          也就是说，
          <span style="color:blue">
           上海人不仅仅在房子上占优，而且在工薪性收入上也不落后。
          </span>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          普通的国企混个副处，一年也有几十万收入。丝毫不比高科技码农少。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果你不讲房子值多少。单单比存款的话，上海人也绝对不少，殷足。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          大赢家。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<blockquote>
<p style="line-height:150%">
<span style="font-size: 16px; line-height: 150%; font-family: 楷体; color: rgb(255, 0, 0);">
           以个人财富而言，上海土著才是过去十五年增长最迅速的群体
          </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
           。虽然外地人“数量”增长更占优。
          </span>
</p>
</blockquote>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          或许还有人会问，唐镇50000/m的房子，500W房子400W税。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          地价和税，是上海市政府拿走没错。滋养得上海本地人滋润万分。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          可我这房子还是价值500W啊。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          是资产我也没亏。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          问题是，500W的
          <span style="text-decoration:underline;">
           唐镇
          </span>
          房子，和500W现金，这绝对不是同一概念的事。
         </span>
</p>
