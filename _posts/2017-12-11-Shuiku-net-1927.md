<p style="line-height:150%;">

         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">

<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">

<blockquote>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">

<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">

<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          国务院是不可以规定“公民不许买房子”的。但是他可以管自己的下属，可以管行政机构。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          可以强迫房产交易中心“不予登记”。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          不予登记的话，意味着银行也不能贷款。而不能贷款，就卡住了99%的买主。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>

<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">

         </span>

<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">

<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">

          绝大多数城市，例如北京，深圳，重庆，他的房产证是被“没收”走的。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">

<span style="font-family: 楷体_GB2312;font-size: 16px;">
          对于北京，深圳等城市，一旦你申请了“银行贷款”。产证是押在银行的。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">

         </span>

<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">

<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          好消息是，一般经过了三个月，六个月。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">

<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">

         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">

<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>

<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          γ，至于某些“产证”扣押，不能做抵押证的城市。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          还是有江湖人士，可以做出全套的借款证明。
         </span>

<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">

<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">

<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">

<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">

<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">

<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">

<p style="line-height:150%;">

<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          七）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          风险的控制
         </span>
</p>
<p style="line-height:150%;">
<span style="z-index:2;left:0px;margin-left:307px;margin-top:184px;width:152px;height:52px;">
</span>
<span style="z-index:1;left:0px;margin-left:254px;margin-top:237px;width:74px;height:52px;">
</span>
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p>
<img class="" data-copyright="0" data-ratio="0.6814621409921671" data-s="300,640" data-src="" data-type="png" data-w="1149" src="{{ '/img/Ok4hZ0tV6r6XYecNkswkcORSpgDXiag0odzkHqqtQZWW507tJoGQq3ib2UedPF6diceTxljs8MAoLMdXlMXBIhGibw.png' | prepend: site.img | replace: '//','/' }}" style=""/>
</p>
<p style="line-height:150%;">
<br/>
</p>
<p style="line-height:150%;">
<span style="font-family: 楷体_GB2312;font-size: 16px;">
          我们回到这张图，逻辑实在是有点复杂。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          在做完债务叠加，
          <span style="color:red;">
           把“残值”压迫到零之后
          </span>
          。你要控制代持人B，还嫌不够。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          因为房地产有一个特性。它是不停“涨”的。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          好比你一套500W的房子，先是借了350W的银行贷款。接着又做了一个300W的“私人抵押证”。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          你算算，负债650W，“代持人”反而倒欠你150W。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          你感到心满意足，敬酒三杯。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          可是你浑浑噩噩，过了几年，毫不关心。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          眼睛一眨，或许这房子升值到1000W了。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          代持人小算盘打打，卖了1000W，扣除650W债务，获利还有350W。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          他依然有背叛的风险。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          而你要做的，是不停关注房产价格。维护，然后“追加债务”。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          从银行系统走，可以追加叠一个“二押”。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          从私人借款走，可以新办一个“抵押证”叠上去。再新欠300W。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<blockquote>
<p style="line-height:150%;">
<span style="font-size:13px;line-height:150%;font-family:宋体;color:#984806;">
           代持方B，怎么会签如此丧权辱国的协议。
          </span>
</p>
<p style="line-height:150%;">
<span style="font-size:13px;line-height:150%;font-family:宋体;color:#984806;">
           借条标明，只要房子卖出，本借条自动作废。
          </span>
</p>
</blockquote>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          除了叠债务之外，另一个控制的方法，是“叠租约”。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          众所周知，含有“超长期”租约的房子，很难很难卖的。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          许多老赖，哪怕房子抵债了。他和朋友签一个长租约，赶也赶不走。还要贴钱送神。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          因此代持的房子，你可以和代持人B签一个“长期租约”。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          中国的法律规定，租约最长20年。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          其实绝大多数人不知道，
          <span style="color:red;">
           租约最长是20年+20年。其中20年收费+20年免费。
          </span>
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          租金，写500
         </span>
<span style="font-size:16px;line-height:150%;">
          ~
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          1000
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          元/月就够了。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          可以写租金已一次性付完，需要高额赔偿金。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          当地税务不高的话，最好到街道里做一个登记。买几张印花税。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          按照中国法律规定，
         </span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">
<li>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
            有登记，优先对抗无登记。
           </span>
</p>
</li>
<li>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
            先登记，对抗后登记。
           </span>
</p>
</li>
</ul>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          登记租约，再交一点税。可以极大增强你合同的真实性。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          第三个方法，则是实际控制。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          千万不要让代持人，住在房子里！
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;color:blue;">
          让他离得越远越好
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          。把所有的月供，银行对账单，电话联系人，统统改到你的名下。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          让他忘记了这套房子最好。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          理论上，“买入”过程中，代持人除了银行签贷款，只需要出现一次。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          因此理想境界是，让“代持人”根本不记得地址。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          整件事，把他处理成局外人最好。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          实际控制可以降低风险。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<blockquote>
<p style="line-height:150%;">
<span style="font-size:13px;line-height:150%;font-family:宋体;color:#984806;">
           最怕代持人铁兄弟，他靠得住。可是有一个不停吹枕边风的婆娘。
          </span>
</p>
</blockquote>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          再下面一个技巧，是“增加证明”。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          你需要
          <span style="color:red;">
           保留全套文档
          </span>
          ，每一笔代支付的首付款，都让代持人另外签一份“代持声明书”。并且拍照录像。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          中国目前法律规定，公证处不得参与此类行为。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          但是在法律实践上，有“五名自然人”见证参与，法律可信度就非常高了。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          因此找五名网友见证签名。并签上“目睹此事为真，x年x月x日”。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          如果一旦代持人翻脸，不承认你是房子真正的主人。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          凭借着全套“首付+月供+代持承诺书”，并且有五个自然人见证。而且房子没有卖掉。你可以起诉。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          法院会“
          <span style="color:red;">
           以事实为依据
          </span>
          ”，判决这套房子真属于你的名下。而且也有了这样的判例。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          只不过银行知晓后，大概率会让你“提前还款”。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>

<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          八）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          利益的分配
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          代持，对于代持人B来说，是“
         </span>
<span style="font-size:27px;line-height:150%;font-family:楷体_GB2312;color:red;">
          有损
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          ”的。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          具体来说，只要贷过一次款，有过贷款记录，哪怕还清，也算“二套”。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          在许多城市，只要买过房子，就不再享有“经济适用房”资格。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          代持，一定是有损的。这点需要和代持人说清楚。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          经常有许多网友问，之前是让亲弟弟代持的。现在弟弟也要买房子，请问如何处理。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          回答是，没法处理。“代持”就一定是受损的。这点要事先讲清楚。生意往来。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>

<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">

<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          房票和贷票，是二个概念。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          某些城市，限贷而不限购。因而只需要支付贷票的费用，付一笔就够。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          干多少活收多少钱。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">

<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">

         </span>

<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">

<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">

<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">

<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
