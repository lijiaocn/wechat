<span style="font-size:16px;line-height:150%;font-family:楷体">

<span style="font-size:16px;line-height:150%;font-family:楷体">
          生产和抢劫
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          水库论坛，最早的几篇微信号是总纲。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          虽然我们有心做一个商业号，庸俗不堪教人如何炒楼发财的账号。将所有的学术性的东西剥离，在“体、用、术”中只讲“用”，不讲“体”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但是有些东西是绕不过去的。在我们最初的三篇之中，还是讲到了这个世界“三观”是什么样子的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          水库论坛，读者是呈指数增长的。目前大约有12000个订阅者，每篇文献的阅读量是4000，也就是三分之一。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          一般微信订阅号的行规是1/10阅读。相比之下，我们的干货是非常非常干的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但是，对于我们最早的几篇“大纲”级文献，成立之初只有约1000个用户，阅读量不超过700，所以很多知识，后来人是不知道的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我们这个世界，主要是“生产和抢劫”二条主线。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          二）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          政治学第一定律
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          人类生物要在地球上住下去，要获得衣食住行金钱食物，方法无非是二种：“生产和抢劫”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你每天按时上班，做遵纪守法的好公民。工人农民护士码农，那你从事的是生产。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          抢劫的事稍微复杂一点。当大官的人，那是第一等抢劫。而如果你是一个普通的老百姓，“一等公民”和“二等公民”之间，还是有所剥削区别。譬如城里人和农民，就是巨大的不平等。而帝都公民又比普通市民再高一点。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          太平盛世，皇权普照，在目前这个时代，纯粹的拦路打劫“此路是我开”已经不多见了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          抢劫，更多地反映为
          <span style="color:red">
           要求政府立法
          </span>
          。竖立对自己有利的法律法规。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          譬如说，今年六月，法国政府刚刚通过了法令。禁止Amazon.com在法国落地。其理由是“捍卫法国传统的街边书店文化”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而这么做背后的实质，就是
          <strong>
<span style="color:blue">
            抢劫
           </span>
</strong>
          了法国购书者，让他们支付更高的价格，来购买实体书店。保护了落后生产力。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          再举个例子，今天寄张发票。但是顺丰说不能寄。因为中国《邮政法》明文规定，所有的票据文件不能通过民营公司寄，只能用
          <em>
           中国邮政EMS
          </em>
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这其实就是
          <strong>
<span style="color:blue">
            抢劫
           </span>
</strong>
          ，又贵又差的EMS在抢劫我。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          抢劫的结果，构成了EMS高层干部的茅台和法拉利。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          了解了“生产与抢劫”的实质，才能够理解
          <strong>
           宏观调控
          </strong>
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size: 16px; line-height: 150%; font-family: 楷体; text-decoration: underline;">
          宏观调控，就其本质而言，就是一场抢劫。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size: 16px; line-height: 150%; font-family: 楷体; text-decoration: underline;">
          就是政治呼声。有政治影响力的人群，要求政府实现倾向于他们有利的立法。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">

</p>

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">

<span style="font-size:16px;line-height:150%">

<p style="line-height:150%">

<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>

<p style="line-height:150%">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我们为什么要反对宏观调控。

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">

</p>
<p style="line-height:150%">
<span style="font-size:21px;line-height:150%;font-family:楷体;color:red">

<span style="font-size:16px;line-height:150%;font-family:楷体;background:yellow;background:yellow">
