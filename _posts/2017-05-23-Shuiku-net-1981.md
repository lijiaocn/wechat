<span style="font-size:16px;line-height:150%;font-family:楷体">
            企业“歧视”绿教徒。把诚信踏实的无神论者都录取了。
           </span>
</p>
</li>
</ul>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          N
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          年之后，
          <span style="color:red">
           请问A企业存活，还是B企业存活。
          </span>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          答案毫无疑问的，肯定是B存活。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          对于“歧视”好坏，本身没有固定的标准。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          有人说猪肉好，也有人哪怕蛋白质摄入不足依然坚持只吃牛羊肉。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          那么，怎么甄别呢。怎样决定“信主”“无神论”哪个更优胜呢。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:微软雅黑,sans-serif;color:red">
          答案就是让企业自由竞争。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          B
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          企业招聘的都是唯物主义，科学研究的踏实人员。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          B
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          企业自然可以研发出拳头产品，高科技产品。抢占市场份额。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而A企业招来的人，只会念经。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你念经有用么。每天朝拜五次有用么。地上会长出牛奶喷泉么。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<blockquote>
<p style="line-height:150%">
<span style="font-size:19px;line-height:150%;color:red">
           所有的私营企业，均不适用歧视原则。
          </span>
</p>
</blockquote>
<blockquote>
<p style="line-height:150%">
<span style="font-size:19px;line-height:150%;color:red">
           因为市场本身已经在优胜劣汰。
          </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
<br/>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
           西方的“政治正确”，歧视只应
           <span style="text-decoration:underline;">
            适用于政府机关，非赢利企业
           </span>
           。
          </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
           圣母们把“歧视”应用到市场化竞争的民营企业，于是造成了大灾难。
          </span>
</p>
</blockquote>
<p style="line-height:150%">
<span style="font-family: 楷体; font-size: 16px;">
<br/>
</span>
</p>
<p style="line-height:150%">
<span style="font-family: 楷体; font-size: 16px;">
</span>
<br/>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          市场本身已经对竞争和
          <span style="text-decoration:underline;">
           成败做出了惩罚
          </span>
          。不需要“看得见的手”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          四）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          结语
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          按照惯例，我们结语时总会大逆转。来一段“以上全错”之类。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:仿宋">
          刚刚经人提醒，原来美团正准备想要“美国上市”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          恍然大悟，那么美团做出一副“政治正确”的姿态。就十分必需，十分必要的了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:250%">
<span style="font-size:24px;line-height:250%;font-family:微软雅黑,sans-serif;color:red">
          你既然想要拿圣母的钱。
         </span>
</p>
<p style="line-height:250%">
<span style="font-size:24px;line-height:250%;font-family:微软雅黑,sans-serif;color:red">
          你就要忍受她们身上的脏。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          虽然是如此的臭不可闻。HR身上每一寸都散发着臭味。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但是“海外融资”，本身也是企业竞争力的一部分。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          只要是竞争，能活下来，就是优胜的策略。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          从这个角度讲，美团炒掉田源。反而是最完美的“市场经济”绝妙证明。
