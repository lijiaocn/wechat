</span>

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          什么叫“不得伤害人类”。不得伤害人类这个定语，即使连人类自身，也分辨不清。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我给你开一道处方药，有一定的副作用。但是药效也属于概率50%的治愈可能。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          请问，这是不是“伤害人类”。该不该让你喝药，该不该强迫他人喝药。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          关于“伤害人类”这个判断，实在太复杂，太缺乏“可操作性”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          西方人在这里跌入一个大坑。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

          <span style="text-decoration:underline;">

          </span>

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">

<span style="font-size:24px;line-height:150%;font-family:微软雅黑;color:blue">

<span style="font-size:16px;line-height:150%;font-family:楷体;color:blue">

</p>
</blockquote>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果你不幸是文科生，或者年龄超过50岁。则你可能不了解“算法”二个字的涵义。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “算法”是计算机业内术语。指你整个的解决方法，一整套解题思路。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          基础科学已经证明，CPU一切指令，可以分解为三类：
         </span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">
<li>
<p style="margin-left:56px;line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            顺序
           </span>
</p>
</li>
<li>
<p style="margin-left:56px;line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            分叉（IF……THEN……）
           </span>
</p>
</li>
<li>
<p style="margin-left:56px;line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            循环（WHILE……WEND……）
           </span>
</p>
</li>
</ul>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          所谓“算法”指的是，把一个现实中的问题，分解成计算机可以识别的“顺序，分叉，循环”指令。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          解决同一件事，有很多种方法。譬如“排序”。就有插值排序法，快速排序法，交换排序法等等。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          算法的优劣，一般有二种评价方法；
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">
<li>
<p style="margin-left:28px;line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            时间复杂性，O（n）,随着样本N的增大，所要消耗的时间成比例增长。
           </span>
</p>
</li>
<li>
<p style="margin-left:28px;line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            空间复杂性，S（n）,随着样本N的增大，所需要的计算机内存比例增长。
           </span>
</p>
</li>
</ul>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果你关心这方面的知识，可以买一些计算机科学的入门书。十分粗浅。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          高二水平就可以了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="margin-left:32px;line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          三）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          算法
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          1997
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          年，IBM公司的“深蓝”计算机。正式击败世界棋王：加里·卡斯帕罗夫。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          当时，这件事也是引起了轰动。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          几乎是吹得没边了。滥竽充数心灵鸡汤都出版了许多本。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而如果你“踏踏实实”，仔细研究一下这件事的话。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “深蓝”的源代码一共有多少。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          答案是，一共才2000行字左右。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          按照一个程序员一天写10行。一个20人的腾讯团队，十天就可以写完。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为“深蓝”的算法简单之至。他其实是“穷举算法”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          也就是说，“深蓝”本身，对整个国际象棋世界一无所知。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          他就是“穷举”。就是内存大。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          当每一步国际象棋，十六枚棋子若是有50个分岔的话。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          则我就50^10，十次方。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我就仗着内存大。和你蛮算。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          其中，“国王”的定价是1000，皇后9，车5，马和象3，小兵1。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          每一个分叉算法中，取“棋力差”最悬殊的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          棋力相等的。选“控制格”最多的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          所以，“深蓝”虽然被吹得神乎其神。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但其实他的“算法”是非常简单的。非常单薄的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          甚至说，我国际象棋只要改了一个规则，你整个的算法就要重写的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          同样道理，“Master”是怎么回事呢。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “国际象棋”被屠掉以后。中日韩的围棋选手，还心存侥幸。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “围棋”的生存优势，是围棋的棋盘特别大。有19*19=361格。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因此每一步下棋，就有361^10，N次方的变化。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这对于计算机是一笔巨大的负担。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          要达到“深蓝”级的彻底穷举，需要多得多的云计算，服务器并联。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          可是在AlphaGo的对阵中，他改进了算法。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          增加了“价值网络”的算法。也就是对整条分叉树进行预判，如果觉得亏子，直接就不展开了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          象虚竹这样“置之于死地而后生”的情况，就不会发生了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          AlphaGo
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          的算法，要比“深蓝”复杂一点。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          肯定不止2000行。估计在5000
         </span>
<span style="font-size:16px;line-height:150%">
          ~
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          10000
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          行之间。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          Still
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          ，在IT史上，10000行的代码算一个大工程么。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          根本算不上。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          微软的WindowsNT有2000 0000行代码。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="margin-left:32px;line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          四）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          算法理解
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:21px;line-height:150%;font-family:楷体;color:blue">
          任何一个能够用算法描述的事情，人类最终必将输给电脑。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          前二天我在京东买了一个“人工智能”的神器。只花了
          <span style="color:#993300">
           9.5
          </span>
<span style="color:#993300">
           元
          </span>
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          他彻底秒掉了99.9999999%的人类。堪称AI界的绝对里程碑。
         </span>
</p>

<span style="font-size:16px;line-height:150%;font-family:楷体">

<br/>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
<br/>

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          人类进化了几万年的文明，曾经一度养活了几万人，几十万人的会计“书算”业。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在这个神器面前溃不成军，一败涂地。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我想说的是，
          <span style="color:red">
           各位目前对AI的成就和分析
          </span>
          ，偏离了一个基本的常识。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">
<li>
<p style="margin-left:28px;line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            你不能基于人类的视角，评论一项AI突破是否伟大。
           </span>
</p>
</li>
<li>
<p style="margin-left:28px;line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            你应该基于“算法”的视角。评论一项AI突破是否伟大。
           </span>
</p>
</li>
</ul>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          AlphaGo
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          击败围棋，虽然在人类史上喧嚣一时。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          可是就算法而言，却幼稚初级。不值一提。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          可能还比不上《Starcraft》《文明》中的电脑智慧。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:12px;line-height:150%;font-family:楷体">
          （下一项科技突破，或许是图像识别。或者全视频人脸识别。）
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这个9.5元的计算器。站在人类的角度，无疑是极其伟大。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          穿越回17世纪的话，再怎么看都是神器。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          可是站在AI史上的话，计算器其实是最简单的东西。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          全部内容二行代码就行了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          再难一点，是“五子棋”。可能要200行代码。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          再难一点，是“国际象棋”。2000行代码。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          再难一点，是“围棋”。5000行代码
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          再难一点，估计是“自动驾驶”。目前据说已经写了200 0000行代码。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为“自动驾驶”的本身算法并不难。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          学过开车的人都知道，“驾驶”真是很简单的技巧。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          无非就是左转，右转，刹车。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          中等智商就能开车。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而谷歌和百度的“无人驾驶”之所以迟迟不能发布成品。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          主要是Exceptional。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          中文翻译为“例外”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          譬如说，你开车开到一半。前面冲出一只野猫怎么办。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          按照标准流程，高速公路应该是干净的。突然有几个土坑怎么办。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          天上掉下来一个冰箱怎么办。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          现代的航空业，“自动飞行”已经到了极其先进“久经考验”的地步。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          几乎所有的航空公司，都号称“自动飞行”可以帮你飞二万英里。只有“起飞/降落”的时候还需要人力辅助。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          可是航空公司毫无办法，机长的工资为310元/小时，副机长150元/小时。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你飞一条国际航线，12个小时，机长的工资就是4000元。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          驾驶员如此之昂贵，号称超过“同等体重黄金”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          那航空公司为什么不取消驾驶员，完全使用“自动飞行”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为还是不能避免Exceptional。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          假设如电影《Sully》萨利机长一样，一起飞就被鸟撞穿了双翼。接下来应该如何危机处理。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:red">
          航空公司赔不起呀！
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我们看“自动驾驶”“自动飞行”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          其和“棋类游戏”的区别在于，他们是在一个
          <span style="color:blue">
           Open
          </span>
<span style="color:blue">
           的环境
          </span>
          中运行的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          对于Open的环境，Exceptional就无穷无尽。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          超出程序所知，超出电脑所知的区域无穷无尽。“自动驾驶”的AI惴惴不安，美国的律师如此厉害；
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          谷歌公司赔不起啊。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="margin-left:32px;line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          五）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          结语
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:21px;line-height:150%;font-family:楷体;color:blue">
          所有能够用算法描述的事情，人类最终必将输给电脑。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这条才是《机器人第一定律》。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我们人类必须很清楚地认识到了这一点。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在人类和机器人的“大较量”之中，关键看“算法”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          只要能用算法描述，必输。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
<br/>
</span>
</p>
<blockquote>
<p style="line-height:150%">
<span style="line-height: 150%; font-family: 黑体; color: fuchsia; font-size: 24px;">
           关键就看一件件件纷纷扰扰的事，怎样被天才程序员归纳成算法。
          </span>
</p>
</blockquote>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          有了算法，该领域人类就一定会失败。时间早晚。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          算法越精妙，算法越逻辑。则超越的时间越早。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          好比AlphaGo，使用了“穷举+预判”算法，而不是深蓝简单粗暴的“穷举”算法。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          时间复杂度，空间复杂度，大大缩减。节约CPU资源。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因此把“击败围棋”的时间提早了十几年。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但是，迄今为止，人类对于“算法”的研究，还处于十分初级的阶段。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          仅限于部分“封闭的，简单估值”的棋类游戏。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:blue">
          仍然是2000行，20000行代码。而不是2亿行，20亿行………
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这离真实的，纷纷攘攘的现实世界，还有无穷无尽的距离。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          象《终结者3》这样的独立机器人，在有生之年或许都不会出现。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          按照业内一般的划分，目前的AI人工智能，仅仅处于“细胞级”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          下一个世代，下一个30年，拼尽全力或许希望可以达到“昆虫级”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:#993300">
          那些热炒AI，夸大AI的资本炒作，恐怕要失望了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:19px;line-height:150%;font-family:仿宋">
          而至于人工智能的“终极成就”呢。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:19px;line-height:150%;font-family:仿宋">
          终极成就则是，
          <span style="color:fuchsia">
           AI
          </span>
<span style="color:fuchsia">
           可以自己判断解决问题。自己写出“算法”。
          </span>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:19px;line-height:150%;font-family:仿宋">
          到了那一天，人类就真的没戏了。
