<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          一)贷款的野望
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          #2460，#2470这二篇，都是几年前写的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          它其实控诉了“50%月还款规定不合理”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          或者说，“只要银行批得出来，这笔贷款就没有问题”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          什么房贷的风险率，面对着全世界最严厉苛刻的审批制度。可以说，“
          <span style="color:blue">
           只要银行肯批，就一定能四五年内全额还完
          </span>
          ”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          紧接着一步，应该是写：“既如此，合理的房贷/月薪比，应该是多少呢”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          50%你说不合理地少。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          60%
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          70%
         </span>
</p>

</p>

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          别停住，脑洞应该继续打开。不打开不算破三观。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          150% ?
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这个数字的回答，才是我们应该“有意识规划”的财务目标。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但今天早上想一想，“中阶篇”不讲这事了。移到后面高阶再讲。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          今天我们继续讲很幼稚，很弱智的技术。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          二)公积金
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          水库论坛写到#2480，也有二百多篇了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          有心兼细心的读者应该会问：“为什么没有公积金”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你仔细地看，从头到尾，二百多篇，连“
          <span style="color:red">
           公积金
          </span>
          ”这三个字都没有提到。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          公积金是一种更便宜的贷款。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          长久以来，从2000年
         </span>
<span style="font-size:16px;line-height:150%">
          ~
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          2011年，我们一直速算，
          <span style="color:red">
           公积金大概比商业贷款“低一码”。也就是0.25%左右
          </span>
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          好比目前基准利率4.9%，但是有大量存量客户，上海土著，都是拿的“七折贷款”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          4.9%*70%=3.43%
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这和公积金3.25%就差得不远。优势仅有0.18%
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          从2012年开始，事情有所变化。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为当时的秀相“宏观调控”。他把利率从7折重新又抬上去了，渐渐地恢复了八五折，九折的口径。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但是原有的“七折客户”，你又不能改合同。否则岂不是要闹翻天。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因此利率出现了“双轨制”。
         </span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            七折用户：3.43%
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            九折用户：4.41%
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            公积金户：3.25%
           </span>
</p>
</li>
</ul>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          从这个角度讲，因为2016年新办的贷款，基本上都是“八八折”“九折”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          所以还是要比公积金贵1.16%，算是个不小的数字。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          今天水库写到了#2480，可是绝口不提公积金。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在当年可不是这样的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          2001年我刚刚入行的时候，“公积金”是绝对的大头。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          当时，几乎所有的《理财周刊》，那些骗死人跳大神的“大师”们，传授的“买房10大技巧”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          开篇第一句话，讲的必然是公积金。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          他们把公积金看得如此重要，以至于忘记了买房。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          不仅仅是学术界，而且民间亦是如此。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          能借公积金的，就绝对不借房贷。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          最好是不贷款，实在实在不行，就仅仅只贷一次公积金。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在2001年时，全上海房贷公积金余额400多亿元。而商品房贷款仅有300亿元不到。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          当时，公积金单笔贷款的上限仅有10万元。补充公积金3万元。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            许多人就是为了用满这10万元，所以他们只买一套14
           </span>
<span style="font-size:16px;line-height:150%">
            ~
           </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
            15W的小房子。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            为了符合公积金贷款条件，而推迟一年二年买房。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            为了使用“组合贷款”，而使得整个流程很慢很慢。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            一辈子只贷过一次10万元公积金，就成了“认房又认贷”的第二套。
           </span>
</p>
</li>
</ul>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:21px;line-height:150%;font-family:黑体;color:blue">
          许许多年以后，他们才认识到，这是多么惨痛的错误！
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          三)公积金的价值
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          公积金最好的用法，就是根本不要去用它。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          为什么，因为这涉及到几个麻烦；
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          1）金额太小，蚊子肉有限
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          2）公积金可能反而更贵
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          3）贷款规划
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          4）其他用法
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我们依次来讲这几个问题。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          首先，我们来回答“蚊子腿到底值多少钱”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          别的理财网站，都总是说“公积金利率低”。但是这个低，到底是多少低。定量计算，只有水库有这个算法。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

          </a>

         </span>

          如果你公积金余额很高，高过了月供很多倍。则我们给你的建议，你做好的做法是“提现”。

<span style="font-size:16px;line-height:150%;font-family:楷体">
          许多人不知道。“公积金余额”其实是可以提现的，尤其是第一条；
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
