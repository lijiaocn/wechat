<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          1）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">

</p>

<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

           </span>
</p>
</li>
<li>
<p style="line-height:150%">

           </span>
</p>
</li>
<li>
<p style="line-height:150%">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

</span>
</p>
<p style="line-height:150%">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">

</p>

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为降价无效，那我干脆把价格挂高。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          价格挂高，有几个好处；
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          1）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在市场上形成：“2年前就卖50000/m”的印象，美化K线图。“xx楼盘一直比隔壁贵”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          2）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在市场上制造New news
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          3）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          告诉对方房东的乐观心态
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          中国的大学里，不教
          <span style="color:blue">
           商业
          </span>
          。“毒教育”
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这让中国的学生，非常地“淳朴”，非常地拿衣服。简单说就是蠢。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          譬如说，中国学生看待“价格”。是幼稚到极点的。他们以为价格就是价格。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          好比一件衣服卖3000元，他们就认为该衣服值3000元。如果明天标5000，他们就认为衣服涨价了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          可能那衣服就一件，竖在门口建立高档形象的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          其实价格是个超级复杂的事情，有多种含义。多种玩法，无数陷阱。原价/现价/促销/套餐/标杆/返奖，Marketing的事，说也说不完。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果你粗懂Bridge桥牌的话，刚开始大家是喊牌。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你喊3黑桃，他喊4方块，对家喊4红心。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          喊牌之间，有固定的套路。透过喊牌向队友传递了大量的信息。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          喊牌不仅仅是争夺坐庄权，另一方面也是沟通与回复。里面信息量极大。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          同样道理，你如果看见业主之前挂1000W，第二天挂1020W，接着挂1025W。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这并不意味着业主就是认为，这房子一定要1025W才能卖。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          业主也是在传递一些信息。这始终仅仅是口头报价。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          四）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          Make new news
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          当市场一旦进入“观望期”，多空僵持。至少有12个月交易清淡。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这段时间内，房东最容易采取的策略。是小幅提价到1100W。但不是直接提上去，而是分好几次提。1020，1040，1060，1080，1100
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          为什么要这样呢。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          哪怕1000W你也卖不掉。“诚意”买家都等候在650W以下呢。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而你要调价到1020，接着1040，这样做的用意何在。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为你在“
          <span style="font-size: 16px; line-height: 150%; font-family: 楷体; color: rgb(255, 169, 0);">
           人工置顶
          </span>
          ”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          Marketing
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          中有一科叫做“Make new news”,指的是你没事要在市场上制造点新闻，刷存在感，刷个脸熟。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          哪怕你的产品完全没有任何改良，没有新的配方。你换个包装，搞个抽奖活动，搞个“百事红罐节日装”，也是必要的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在房地产市场，这相当于“人工置顶”，顶贴，顶。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为你的讯息，在中介数据库里会
          <span style="text-decoration:underline;">
           沉底
          </span>
          的。一套房源，放了二三个月，就会沉到小本本最后一页。从此中介很少关注，有客户也不再推介，基本属于打入冷宫的状态。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          那你就需要“顶”。一般在市场中，刚出来7天之内的房源，是中介花力气推得最多的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          中介也知道房子不好卖。所以“新”房源就意味着希望，“新”状态就意味着希望。(譬如你和中介说
          <span style="text-decoration:underline;">
           全新
          </span>
          翻修，其实就新买了个洗衣机)。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我们是诚心想卖，所以我们会小幅加价。加价的幅度不多，象征性的1000到1020，而绝对不会是1000到1300.
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但我们会小幅加很多次。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          一方面，增加“顶贴”的次数。增加被售出的几率。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          一方面，显示房东非常有信心。大力看好。还不停地在加价。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          一方面，给傻空增加心理压力。“二个月不见，又涨了20W”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          五）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          逼空之路
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          楼市呈现非常明显的“季节性现象”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          一般而言，是几个步骤：
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          1）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          单数年春节后开涨，三四个月内+50%涨幅
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          2）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          政府出调控政策。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          3）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体;background:yellow;background:yellow">
          因为宏观调控，所以房价飞涨
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          。宏观调控
          <span style="text-decoration:underline;">
           托实
          </span>
          涨幅，使下跌不可能。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          4）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          市场因为“心理原因”，而不是实际原因进入观望期。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          5）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          冰冻6
         </span>
<span style="font-size:16px;line-height:150%">
          ~
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          12
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          个月。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          6）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          双数年一般是补涨，落后洼地追平。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          7）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          休息足够，买力积蓄，迎接下一轮暴涨
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          中国的房地产市场，已经
          <em>
           扭曲扭曲扭曲
          </em>
          到了骇人听闻的境地。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          以至于他清楚地划分为“观望期”和“窗口期”。时间大致是20个月+4个月。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在正常交易的资本主义国家，是没有这回事的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<blockquote>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
           学院派的分析研究报告，是永远不会和你说这些事的。
          </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
           只有第一线的炒家，才能教你。
          </span>
</p>
</blockquote>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在漫长的“观望期”中，你几乎是不可能按照“市场价”成交的。或者在图形中，不可能按照蓝线成交。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
<br/>
</span>
</p>

<br/>
</span>
</p>
<p style="line-height:150%">
<span style="line-height: 150%; font-family: 楷体;">
          他只有非常短暂的几个点，蓝线和红线是相交的。也就是
         </span>
<span style="line-height: 150%; font-family: 楷体; color: red;">
          傻空洗面革新，终于肯承认接受多军是估值正确的
         </span>
<span style="line-height: 150%; font-family: 楷体;">
          。
         </span>
<br/>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这几段“窗口期”非常之短。一般也就3
         </span>
<span style="font-size:16px;line-height:150%">
          ~
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          4
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          个月左右。因为政府闲不住，看得见的手总喜欢乱摸。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          一调控，实质内容不重要。关键是傻空“心理”又变成了再坚持一下。然后又是漫漫观望。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          宏观调控搞了十年，上海楼市涨了十年。但其实它是在“没有充分换手”的基础上上涨的。任何一个价位都没有夯实。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
