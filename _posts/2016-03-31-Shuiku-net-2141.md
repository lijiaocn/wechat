<p style="line-height:150%">

<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<span style="font-size:16px;line-height:150%;font-family:楷体">

<span style="font-size:16px;line-height:150%;font-family:楷体">

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<li>
<p style="line-height:150%">

<p style="line-height:150%">

</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<span style="font-size:16px;line-height:150%;font-family:楷体">

<span style="font-size:16px;line-height:150%;font-family:楷体">

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为“压价效应”的存在，意味着永远有一群房东“脑子不正常”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          明明是一千万的房子。他偏偏要920W抛。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          明明是999个业主都挂的1000W报价，但偏偏就是一个奇葩业主挂920W。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在正规的教科书中，这样的“蠢货”房东是不存在的。这等于送钱效应和bug。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果一定要存在，则教科书会告诉你，会出现“套利”行为。把这个8%的差价给吃光。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          具体的说法，即是“
          <span style="color:red">
           我为何不920W买进，转手1000W卖出
          </span>
          ”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          问题就出在了这里。问题就在于，你不能“920W买进，1000W卖出”！
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为教科书没有考虑过这种情况，因为税费+限贷+限购的存在，买卖一次的交易裸成本，接近了18%
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这么高的“交易成本”，在以往的非中国的教科书中，从来没有发生过。也从来没有西方学者研究。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          他们不认识伟大的秀相。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在秀相的“高压调控”之下，整个市场的“交易频率”达到了匪夷所思
         </span>
<span style="font-size:27px;line-height:150%;font-family:楷体;color:blue">
          低
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          得不能再低，简直就是拒绝交易。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          整个小区，挂牌量不超过1%是常有的事。许多小区三五年都没有一套成交。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          所以，在“极低低低低地频”交易市场中，就出现了8%“压价效应”的奇异现象。
         </span>
</p>
<blockquote>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:blue;background:yellow;background:yellow">
           你卖楼只能按照“最低价”来卖，不能按照“市场价”来卖。最低价=市场价92%
          </span>
</p>
</blockquote>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这个只有
          <span style="text-decoration:underline;">
           活学活用
          </span>
          经济学的人才会理解。熟读书的蠹虫是不明白的。
         </span>

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          五）压价效应下的卖楼
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          当有“压价效应”存在时，整个“售楼过程”变得很麻烦。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为绝大多数的业主，大家都是1000W的报价。你横比竖比，和隔壁前后楼盘比较，心中也认可了这个1000W的价值。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          可如果一旦你到市场上去卖，前面有一个SB的业主（大概率是温州人），他非得要挂920W．
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          那你怎么办呢。你如果想抢跑道。你就得比他更便宜，你得挂919W元。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果你不抢跑道，你就需要时间。需要时间把它消化。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          其实，你可以这样想。“压价效应”和“笋盘”这二个概念。是完全相辅相成一一对应的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          水库论坛，一直都在说“
          <span style="color:red">
           笋盘七折，非笋勿入
          </span>
          ”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但是如果你换一个角度想呢，假设市场上全部都是“水库网友”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          每一个人都确定了“非笋勿入原则”。除非是市场上的七折，否则绝不买入。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          每一个人。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          那么，对于业主来说，他又如何才能抛售自己的房产呢。“压价效应”岂不是变成了-30%。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          六）好日子和旺季
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果定义：房价=全小区10套平均报价

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          定义：最低价=最低那套售价
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          压价：最低价/房价

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          当你卖出时，你一定是全小区（乃至全地区）最便宜的一套。
         </span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            对于地标性知名楼盘，“最低/平均”的差距较小。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            当套利者/炒家存在时，“最低/平均”的差距较小。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            在旺季中，“最低/平均”的差距较小。
           </span>
</p>
</li>
</ul>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          好比如果你去郑州商品交易所兜售小麦，同时可能有100000个农场大队在抛售粮食。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在这个时候，“售价最低/平均价”这个比值就会非常接近99.9999999%.
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果有哪个农民比别人便宜0.02元，他就飞快地在几秒钟内被人扫掉。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而相反的道理，当秀相把“交易成本”加到了18%。则相应的，
         </span>
<span style="font-size:21px;line-height:150%;font-family:楷体;color:blue">
          衍生的
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “压价效应”也会被放大。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          会扩张到7
         </span>
<span style="font-size:16px;line-height:150%">
          ~
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          8%的比值。你抛售时，绝对抛不到市场价。一定得比市场价显著低才行。
          <a name="_ftnref1" style="font-size: 10px; text-decoration: underline;" title="">
<span style="line-height: 150%; font-family: 楷体; font-size: 10px;">
            [1]
           </span>
</a>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          前二年京沪楼市的买家，跑到市场上看一下，如果是“市价盘”，基本上就懒得出手了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          成交如此清淡。只有你
         </span>
<span style="font-size:21px;line-height:150%;font-family:楷体;color:red">
          显著
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          低于周边正常市场价。才会吸引购买。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这也就是我们为什么要写“三重顶”。为什么特别强调，在旺季一定要抓住那几个月，甚至不惜把租客赶走，一定要保持品相靓丽加紧卖楼。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为在旺季中，“最低价/平均价”也会不断趋近于1。那些显著
          <strong>
           低于市场价
          </strong>
          的盘会被扫掉。
         </span>
</p>
<p style="line-height:150%">
<span style="text-decoration: underline; font-size: 16px; line-height: 150%; font-family: 楷体; color: red;">
          旺季中，你终于可以按照“市场价”来抛售你的房产。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>

<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          七）市场价的意义
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          傻空是一群脑垂体发育不太完全，当然人品RP也非常恶劣的群体。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你跟他说了“最低价/平均价”的道理。傻空想象的，不是指责秀相18%的交易成本恶政。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          相反，他们会辛灾乐货的说，“为什么你不认为，你的房价本来就是920W呢”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          傻空们振振有词，既然你这房子只能够按照920W卖掉。那么你的“价格”就是920W。一千万
          <em>
           从来没有实现过
          </em>
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          那么，这个1000W还有什么意义呢。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:red">
          答：市场的平均价，80%最主流的报价，意味着“
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体;color:blue">
          一定可以实现的价格
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体;color:red">
          ”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我们要告诉傻空的是，这个1000W的意思，就是“一定可以实现”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          那么
         </span>
<span style="font-size:16px;line-height:150%;font-family:微软雅黑,sans-serif">
          他就一定会实现
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          判断“未来”是最困难的事。谁也说不清楚明天的涨跌。金融市场的看多和看空，是最最折磨人的事。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但我们可以告诉你的事，一旦“主流”出到了1000。则这个
          <span style="color:blue">
           1000W就一定会出现在时空的长河之中
          </span>
          。
         </span>
