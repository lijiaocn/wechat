<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:blue">

</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">

<p style="line-height:150%">

           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            “要开人大了，改法律土地私有。然后就可以开征物业税了”。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            “虽然要征物业税，但是土地变私有的了。我想房东也是乐意的”。
           </span>
</p>
</li>
</ul>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:blue">
          正确的逻辑：
         </span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            没有土地才需要交税，有了土地才不用交税。“正向”资产。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            能保护你的，是宪法。而不是投机取巧。
           </span>
</p>
</li>
</ul>

<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          一）社会契约论
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          18世纪，卢梭提出了“
          <span style="color:red">
           社会契约论
          </span>
          ”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          之前的二千多年，都是农业社会。实行的是“绝对君主制”。皇帝拥有无限的权力。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          到了工业革命之后，需要一种新的社会组织方式，以及生产力理论基础。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “社会契约论”是现代社会的运转基石。几乎所有的
          <span style="text-decoration:underline;">
           现代化文明国家
          </span>
          ，都依赖于此。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “社会契约论”讲的是，全社会依靠一系列的“契约”而运行。
         </span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            无代表权不纳税，国王不可以随便征税。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            “风能进，雨能进，国王不能进”
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            三权分立。五族共和。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            罗伯特议事法则。
           </span>
</p>
</li>
</ul>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这里面，对每一类人群都进行了约束。无论是国王，贵族，人民。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你一间茅草屋，再卑贱的贱民；
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但是“风能进，雨能进，国王不能进”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          二）产权
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          随着“社会契约论”而来的，是“产权”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为你要签订契约，你首先要界定“你我”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          哪些是你的，哪些是我的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          有了“你我”，就有了“产权”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          虽然经济学的各大流派不同。但有一点大家是共识的。“产权”永远是第一位的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          人只有为自己争取利益时，才有最大的激励能动性。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          凡是大锅饭，一定偷懒的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          产权是现代化一切分工合作的基础。有了清晰的产权，才能签订合同，才能结算分帐。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          工业和商业社会，千千万万数不胜数的上下游采购销售，人力劳务。其背后都是“合同”，都是产权，都是契约。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          离开了合同，现代社会一天也活不下去。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-family: 楷体; line-height: 150%;">
          没有了“合同”，则一切工商业活动都无法展开。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          整个社会的生产力，会徘徊在非常低的水平。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          类似于北朝鲜或者非洲黑叔叔饥饿贫穷。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
<br/>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          三）财产税
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          好了，现在我们可以立即推导出，所谓的“财产税”不存在。
         </span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            世界上不存在“因为你拥有财产而需要交税”的说法。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            世界上只有“根据财产金额而计算税”的说法。
           </span>
</p>
</li>
</ul>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          中国的文科生小编，非常地文盲。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          他们既没有学过任何真正的知识，类似于白痴一样。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          却又忍不住指点江山，对什么都要评论。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          世界上根本不存在“拥有财产而要交税”的说法。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为财产本身是“产权”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你的，我的，划分得清清楚楚。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          政府的，民间的，划分得清清楚楚。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果你说“财产交税”的话，这就等于侵犯了产权。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我就无法界定，哪些才是“我真正拥有的”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “产权”制度混乱了以后。整个社会就无法运行。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          首先就没有人干活了。如果你可以抢劫我的话，中国会迅速地沦为北朝鲜。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          其次整个社会都没法签订上下游合同了。我不知道你会不会价外加价，谁敢和你做生意。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          好比你在饭店里吃了一碗饭，付了20元。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你刚想离开，被店员拉住了。“客官，请再付20”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你又付了20，“客官，请再付40”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你又付了40，“客官，请再付80”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          …………这事是没完没底的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          什么叫做合同。合同就是执行完了之后，
          <span style="color:blue">
           权利义务划分清楚。二不相欠
          </span>
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “财产税”本身是和“产权”概念冲突的。无法存在于现代社会。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “付了20，再付20，再付20，再付20”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “财产税”就其自身而言，也是根本逻辑不自洽的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          四）契前税
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          接着有人问，“不对呀，生活中有
         </span>
<span style="font-size:16px;line-height:150%;font-family:仿宋">
          买了再付费
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          的例子”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你买了房子，还要付物业管理费。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你买了汽车，还要付“车船使用税”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
<br/>
</span>
</p>

<br/>
</span>
</p>
<p style="line-height:150%">
<span style="font-family: 楷体; line-height: 150%;">
          这些人的诸类提法，只能说明他们全部都是“法盲”。
         </span>
<br/>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          为什么，因为他们提的全部都是“契前税”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            “要开人大了，改法律土地私有。然后就可以开征物业税了”。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            “虽然要征物业税，但是土地变私有的了。我想房东也是乐意的”。
           </span>
</p>
</li>
</ul>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="color: blue; font-family: 楷体; line-height: 150%;">
          正确的逻辑：
         </span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            世界上根本不存在“财产税”。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            没有土地才需要交税，有了土地就不用交税。“正向”资产。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            能保护你的，是宪法。而不是投机取巧。
           </span>
</p>
</li>
</ul>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          哥哥每天看四五个小时楼市信息。不停地看见文盲法盲们辩论；
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “因为土地不是私有的，所以不能征物业税”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          SB，房产证总是你的吧。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我开征“房产证持有税”好了，每年1W。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          什么，房产证遗失了。那还有“房产证中央电脑登记税”。不要挑战笔杆子的能力好么。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          之前社会上对于“房产税”的分析，几乎全盘都是错误的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:blue">
          关于目标，方法，困难，全部都是错的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          逻辑层层反。几乎读不下去。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这所有谬误的集大成者，是《南方周末》在2010年发表的一篇《征收房产税·凭什么为什么》
          <a name="_ftnref1" style="font-size: 10px; text-decoration: underline;" title="">
<span style="line-height: 150%; font-family: 楷体; font-size: 10px;">
            [1]
           </span>
</a>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这篇文章，因为《财经》等杂志转载。获得了极大的阅读量。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但是他整篇都是错误的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          文中说了屌丝如何苦，屌丝如何穷，屌丝买不起房子。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          最后得出的结论，问房东征物业税。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          技术难题是，土地非私有。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          实在是看不下去，整篇文章从立论，到方法，到困难，几乎所有的分析都是错误的。
         </span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            土地非私有，完全可以开征“房产证持有税”。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            错误的错误，在于“持有税”的说法不存在。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            错误的错误的错误，房东没责任付钱养活你。
           </span>
</p>
</li>
</ul>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          从《南方周末》《财经》等邪恶媒体，我们完全可以看到左翼文青是如何毒害这个国家的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          幸好中国没有搞民主制度。否则这股流毒害死了欧洲，害死了美国，害死了美丽岛。蠢蠢又想害中国。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          为什么不能开征房产税。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为房东不是你爹，房东没义务养活你。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为按劳分配。你想要更好的生活，你就应该去劳动。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为宪法！
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为宪政和法制，保障着善良人不受屌丝的侵害，不受抢劫的侵害。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为宪政和法制。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为善良，正义，光辉！
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          南方周末，你这个不要脸的寄生虫！
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
