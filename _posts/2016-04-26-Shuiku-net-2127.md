</p>
<p style="line-height:150%">

<p style="line-height:150%">

</p>

<span style="font-family: 楷体; line-height: 150%;">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而对于空军，还有诸多观望者，他们其实想要问的另一个问题。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          何时才是尽头，天空才是尽头”？
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          二）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          涨幅
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我们要说的是，以上二者的说法都不对。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          为什么，因为京沪楼市有“隐患”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-family: 楷体; line-height: 150%;">
<br/>
</span>
</p>
<p style="line-height:150%">
<span style="font-family: 楷体; line-height: 150%;">
<img data-ratio="0.6564748201438849" data-s="300,640" data-src="" data-type="png" data-w="" src="{{ '/img/Ok4hZ0tV6r4darVHf6ERBd0tou91xbMyUhqM7rPnKeB6hyOGQEgYrtNllUnZmYR50jZfWQ5XLnIQYHia8exiah4g.png' | prepend: site.img | replace: '//','/' }}"/>
<br/>
</span>
<span style="font-family: 楷体; line-height: 150%;">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “房价”为何会上涨。这是从2003年第一次大涨之后，多空讨论不休十几年的事。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          当时的空军领袖说，“看来看去，除了
          <span style="color:blue">
           滥发纸币
          </span>
          一条，其余都不支持房价上涨”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          社会的生产力肯定是不断进步的。任何一件物品的制造成本，现在肯定比五年前低。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          科技更先进，机器更大型，材料更密致，施工更合理。科技本身就在导致成本不断降低。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而一件商品，从“制成后”还能够不断地“升值”。甚至升上几倍，几十倍。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          那除了
          <span style="color:blue">
           货币注水
          </span>
          ，没有别的解释了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          毕竟，价格=房子/人民币含金量
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
<br/>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但是，我们看回示意图2。在图中蓝线代表“物价”。物价涨得并没有房价高。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在过去16年，房价涨了16倍。
         </span>
</p>
<blockquote>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
           但是物价指数，基本上是“改革开放迄今，36年涨了64倍”。
          </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
           每一届12年4倍。
          </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
           每6年翻翻
          </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
           每年12%
          </span>
</p>
</blockquote>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你对比一下的话，（京沪）房价是每4年翻翻。每年19%
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而物价是每六年翻翻，每年12%
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这其中就有一个
          <span style="color:red">
           缺口
          </span>
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          按照正常的比值，假如房价跟随物价上涨，纯粹是一个货币现象。
         </span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            房价十六年应该涨了6倍
           </span>
</p>
</li>
</ul>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          6
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          倍，而不是16倍。那么，这额外多出来的倍数去哪里了呢。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我们想了很久，绞尽脑汁，用了各种理由和借口，去圆这个“理论漏洞”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
<br/>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<br/>
</span>
<span style="line-height: 150%; font-family: 楷体;">
          如
         </span>
<span style="line-height: 150%; font-family: 楷体;">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在过去的2000
         </span>
<span style="font-size:16px;line-height:150%">
          ~
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          2016
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          年，正常的物价涨幅应该是6倍。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但是，房价的涨幅是16倍。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在16倍中，正常的涨幅应该是10倍。还有额外的60%是因为秀相的“宏观调控”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;background:yellow;background:yellow">
          因为宏观调控，所以房价飞涨
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          。宏观调控导致了额外的+60%涨幅。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这件事，我们在之前的《
          <a data_ue_src="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=403530885&amp;idx=1&amp;sn=53c615c338024587afaa293a9f90eb23&amp;scene=21#wechat_redirect" href="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=403530885&amp;idx=1&amp;sn=53c615c338024587afaa293a9f90eb23&amp;scene=21#wechat_redirect" target="_blank">
           吃毒药涨内力
           <sub>
            #950
           </sub>
</a>
          》中已经说过了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但是，你还是有一块是不能解释的。就是图中的红色部分。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          红色部分，还有+60%的涨幅。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这一段的解释，或许是“大城市化”。也就是过去十六年中，由于秀相的“国进民退”，全国资源大量集中于政治特权性城市。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          所以导致了中心城市人口，资金，产业，投资的高度密集。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          四线城市萧条，地级市失血，省会半梦半醒，而一线城市狂飙。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          三）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          安全区域
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在所有的涨幅中，只有“货币增发”原始的这6倍是可靠的。其余都会遭受攻击。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          很多人都在问，“房价何时下跌”。其实这是一个伪命题。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为他包含了二个部分；
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          1）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          房价永不下跌
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          2）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          涨得过分了，就会下跌。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          譬如说，我们继续看示意图4
         </span>

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          换算成时间，这就是“二年”。曾经以为蠢不可及的CEO盘，买入也就套二年。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          二年之后，你这价就是市价。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          四年之后，你获利30
         </span>
<span style="font-size:16px;line-height:150%">
          ~
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          40%
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:blue">
          可见，哪怕你买入一项非常糟糕的资产。在通货膨胀的大环境下，也能秒杀现金。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果你草率地仅是输了二年时间，99%人完全都扛得住。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          那么，如果你这些资产“非常非常非常”糟糕。糟糕到哪怕连你都输时间输不起了。它得有多糟糕呢。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我们草率地估算，大约是12年。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          12
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          年，是香港楼市套牢的跨度。
         </span>
</p>
<p style="line-height:150%">
<span style="font-family: 楷体; line-height: 150%;">
<img data-ratio="0.44015444015444016" data-s="300,640" data-src="" data-type="jpeg" data-w="518" src="{{ '/img/Ok4hZ0tV6r5qZUcrCTS0ybwGb8xub3icbIx3dgk6AZChRlK7yGFLoGSmLsHqVZFuhicWXvT2n9BHMq698ag7GiaRg.jpeg' | prepend: site.img | replace: '//','/' }}"/>
<br/>
</span>
</p>
<p style="line-height:150%">
<span style="font-family: 楷体; line-height: 150%;">
          如果你看回香港房东的话，大约“浪费”了十二年的时间，十二年的人生，十二年的青春。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          12
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          年之后，房价终于脱离了蟹货。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果以“12年耐心”为估算的话，则十二年的通涨一共是400%
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          也就是市场可以接受比价值高400%的泡沫。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          六）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          战术
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          上海房子其实不值那么多钱，不值100000/平米。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          对于这件事，多军都心知肚明。无论房租，同类，细分，跨省，新区，等等众多指标都显示背离。以后我们会讲到。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但是，“虚高”和“会不会”涨，绝对是二个概念。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          按照欧美同类对比，A股大概只值600点。但他今天扔在上串下跳。每天还在讨论涨停板赚钱。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<blockquote>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:#7030A0">
           在过去的十六年，我们连续押了16把大。赚到盆满钵满。
          </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:#7030A0">
           如果你要问2017，2018年，估计我们仍然是押大。
          </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:#7030A0">
           “
          </span>
<span style="font-size:16px;line-height:150%;font-family:楷体;color:#7030A0">
           虚高”和“涨不涨”绝对是二码事。
          </span>
</p>
</blockquote>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          京沪楼市的唯一问题，是涨得太快。透支了几年涨幅。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但这绝不意味着明年就不涨了。从各种具体细微之处可见，明年仍是大涨年。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为除了战略判断之外，你还有战术判断。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          楼市如果真的到顶，崩溃之前会有大量战术迹象，会有各种细节端倪。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          战术是什么。京沪到底可以涨到哪一个价位见顶。
         </span>
</p>

<span style="font-size:16px;line-height:150%;font-family:楷体">
