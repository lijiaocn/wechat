想用 AI 颠覆传统教育，这家公司正在招募人才

发布日期：2017-10-19 06:23:05 +0000

  • 前一篇
    • internet / 小道消息
  • 后一篇

    • 2017 年 7 月 26 日，流利说在北京举行发布会，宣布完成近亿美金的 C 轮融资。



    流利说于 2012 年在杭州成立，创始人是王翌、胡哲人和林晖。流利说团队刚成立那会儿，在杭州贝塔咖啡遇到王翌（他是前 Google 产品经理）好几回，就英语学习这事儿还交流过。但通过一个 App 学英语这事儿有多大想象力，说实话看不准。等产品出来了，体验了一下，体验不错，还热心的推荐给了几位投资人，有投资人看完说，是不错，可是怎么赚钱呢？那个时间点，都不知道在线教育能怎么变现。


    一转眼，这个团队的旗舰产品「英语流利说」已经拥有 5000 万用户，2016 年 7 月推出的全球第一个 AI 老师「懂你英语」也已经拥有超过 60 万用户，这是个付费产品。


    怎么定位这家公司呢？「AI + 教育」，要用 AI 颠覆传统教育。


    流利说创始团队和核心成员很「豪华」，有好几位 Google 前员工，有微软、Facebook 等全球知名人工智能公司和实验室的顶尖科学家和一流的工程师，团队有较深的学历背景和扎实的研发能力。


    总部在上海，目前在杭州、南京、硅谷设有分公司。团队文化也很好，「弹性工作制，全员 Mac 办公」这种自然不在话下。


    流利说团队正在进行大规模招聘，如果你精通如下一个或多个领域，并且具有超强的学习与研究能力，如果你也曾感概教育的低效率与不公平，不妨考虑加入这个团队，通过技术帮助更多的用户。



    招聘职位



    资深后端工程师 （上海）


    • 具备 Ruby / Golang 开发经验，计算机基础知识扎实，包括不限定于操作系统、计算机网络、数据结构和算法；

    • 熟悉关系型数据库，如 MySQL 等，熟悉 NoSQL，如 Redis 等。


    薪资范围： 20K - 40K


    补充说明：团队以 API 为主，但非常鼓励全栈，只要你想，可以有很多写前端的机会（比如 React），在这里你可以「折腾」百亿级数据库表。


    很早就使用 Docker、Kubernetes 等前沿技术来构建智能化自动化运维体系，摆脱繁复开发和部署流程走上精益之道。



    资深 Android 工程师 （上海）


    • 熟练掌握 Java 或 Kotlin，熟练使用 Android Studio 及相关调试工具；

    • 热爱移动 App 开发，对新事物敏感，喜欢探索，有较强的创新意识与责任感。


    薪资范围：20K - 35K


    补充说明：流利说开发团队一直走在 Android 开发最前沿，紧跟潮流的 Android 团队有些项目已经大规模使用 Kotlin开发，项目早已成熟应用组件化架构并且近期也开始研究插件化。



    资深 iOS 工程师 （上海）


    • 基础扎实，数据结基础知识掌握牢固，有良好的编码风格；

    • 熟练掌握 Swift 语言，有丰富的 iOS 开发经验，熟练使用 Xcode 及相关调试工具。


    薪资范围：20K - 35K


    补充说明：从 Swift 1.0 开始实践到今天，是国内为数不多的全 Swift 开发的团队；团队成员每年都活跃在 WWDC 的前线。



    自然语言处理工程师 （上海、杭州）


    • 熟悉机器学习在 NLP 领域的应用，能利用机器学习技术改善 NLP 相关技术的效；

    • 精通某一种编程语言（Go、C++、Java、Python 等），有一定的项目经验；

    • 有测评系统、自动作文批改、机器翻译、纠错算法研发经验者优先；

    • 熟悉文本分类、语言模型、实体识别、知识图谱抽取和推理、问答系统设计等 NLP 相关知识；


    薪资范围：25K - 50K


    流利说的算法团队由首席科学家林晖带领，并由多位来自来自 Google（首席算法工程师孙怿与首席算法架构师张潇君曾就职于 Google）、微软等全球知名人工智能公司和实验室的顶尖科学家和一流的工程师组成.


    每年都有众多团队成员活跃于 ACL、Interspeech、KDD、ICML等国际学术会议，团队多项成果发表在国际一线期刊。



    大数据开发工程师 （上海）


    • 扎实的计算机基础, 至少一年大数据平台相关经验；

    • 熟练掌握如下最少一个组件: Hadoop/Spark/Kafka/Presto 等, 具备问题定位/解决能力, 有 HBase/Cassandra 等分布式存储相关经验加分, 有开源社区贡献的加分, 有 AWS 平台经验加分。


    薪资范围：20K - 30K


    补充说明：这里有数千万学习用户的海量数据，数据收集、计算、存储、资源调度、数据应用等各种问题等你来挑战；Hive、Spark、Presto、Flink 任选 。



    测试开发工程师 （上海）


    • 熟悉常用接口测试工具、可以独立搭建接口自动化测试平台；

    • 熟悉常用移动客户端自动化测试技术及相关编程语言；

    • 了解测试基础理论，熟悉常用测试技术方法，有较高的质量意识。


    薪资范围：15K - 25K



    跟流利说 CTO 胡哲人做了个简短的交流，他们团队的风格至少还包括这些：


    • 可能是创业公司中 Code Review 最认真的团队，每个工程师的代码至少被一个人 Review 过；

    • 热爱写测试。

    • 用 Slack 作为沟通、运维、交流的工具，用 Phabricator 进行项目管理；

    • 是一群喜欢研究各种新语言、新技术的工程师，如 Swift、Go、Kotlin、React，而且很喜欢把新技术和语言应用到工作当中；

    • 鼓励开源热衷奉献，团队的 GitHub 开源项目有近万个 Star，屡上 GitHub Trending 榜： https://github.com/lingochamp

    • 每年国内外顶尖技术大会 如 WWDC、 Google I/O、AWS re:Invent、Spark Summit、QCon 总有流利说技术团队成员的身影。
