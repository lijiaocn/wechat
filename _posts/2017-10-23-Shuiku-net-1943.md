<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          紫檀这玩意，三十年长一厘米，千年不腐，木质紧密。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          但是，生鲜，水果，牛奶，蔬菜，就属于“
          <span style="line-height: 150%;font-family: 楷体_GB2312;color: red;font-size: 18px;">
           极易腐烂
          </span>
          ”的食品。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          在运输条件不好的19世纪，牛奶从美国中部的农场，运到大西洋边的纽约。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          耗时可能要一周，甚至更多。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          腐烂变质不能喝。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<blockquote>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
           紫檀不会扔进河里。但是牛奶会。
          </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
           当售价赶不上运费。
          </span>
</p>
</blockquote>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;color:blue;">
          更何况，在21世纪的今天，如果你“不幸”生存在欧美等国家，你还要面临一些“控诉”费用。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          如果小朋友喝了你的牛奶，拉肚子怎么办。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          老奶奶吃坏了住院，索赔怎么办。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          圣母敲诈勒索怎么办。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          赚着卖白菜的钱，操着卖白粉的心。欧美“吸血律师”，动不动罚你赔偿几百亿美金。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          这些“法律”成本，远远超过了牛奶本身的售价。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          三）要素
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          经济学有几大要素，价格=f
          <span style="line-height: 150%;font-family: 楷体_GB2312;font-size: 12px;">
           （时间，地点，人物，事情）
          </span>
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          学理工科的人，是极其严谨的。和Karl Marx的嘴炮不同。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          f
         </span>
<span style="line-height: 150%;font-family: 楷体_GB2312;font-size: 12px;">
          （x）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          只要任何一个要素变了，价格都会变。
         </span>
<span style="font-family: 楷体_GB2312;font-size: 16px;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<blockquote>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
           冬天的西瓜，夏天的草莓。
          </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
           都市里的牛奶，农场里的快递。
          </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
           要素的任何变化，都会导致价格剧烈波动。
          </span>
</p>
</blockquote>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          马克思主义的不要脸之处，在于他混淆了“时间和空间”。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          你一边说，家里面的小孩，饿得哇哇哭。营养不足，没有牛奶喝。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          一边又说，农场黑心的资本家，把牛奶倒在河里。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          “冰箱里的牛奶”和“农场里的牛奶”，
          <span style="text-decoration:underline;">
           并不是同一件商品
          </span>
          。你这样
         </span>
<span style="font-size:19px;line-height:150%;font-family:楷体_GB2312;color:red;">
          偷换概念
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          ，
          <span style="font-size:16px;font-family:楷体_GB2312;color:red;">
           不要碧莲
          </span>
          么。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">
<li>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
            牧场里的牛奶，我可以免费送给你喝。
           </span>
</p>
</li>
<li>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
            可是冰箱里随手有牛奶，我真的做不到。
           </span>
</p>
</li>
</ul>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          把“农场里”的牛奶，运输到大都市来，是一个专门的行业。英语叫做Logistic，我们叫做“物流业”。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          目前在GDP中的份额，大约占9%
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          马克思主义长期把“物流业”痛斥为第三产业。以为它不创造价值。这是极歪的三观。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          “物流”是商品中不可或缺的一部分。穷疯了头，才会只看“
          <span style="font-size: 16px;line-height: 150%;font-family: 楷体_GB2312;color: rgb(61, 167, 66);">
           产地价格
          </span>
          ”，呸！
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size: 16px;line-height: 150%;font-family: 楷体_GB2312;">
          冰箱里的牛奶，
         </span>
<span style="font-size: 24px;line-height: 150%;font-family: 微软雅黑, sans-serif;color: blue;">
          成本=牛的奶+冶炼加工+运输到冰箱
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          四）信息成本
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          随着中国经济的发展，逐渐由一个农业国，转型成为工业国。“运输业”日渐庞大。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          目前中国机动车保有量，超过了3亿辆大关。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          按数量级换算，全国至少有几千万人从事“司机”物流行业。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          因此近期的文献，渐渐地不谈“服务业”。
         </span>
<span style="font-size:19px;line-height:150%;font-family:微软雅黑,sans-serif;color:blue;">
          物流司机，也被视为“工人”
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          ，成为劳动人民的一部分。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          老古董们，开始习惯了工业化生活。
         </span>
<span style="font-size:16px;line-height:150%;font-family:微软雅黑,sans-serif;color:fuchsia;">
          觉得物流和炼钢一样，也算劳动了。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          可是，另有一块成本，秀相白左们无论如何也不肯承认。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          冰箱里的牛奶：
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:24px;line-height:150%;font-family:微软雅黑,sans-serif;color:blue;">
          成本=牛的奶+冶炼加工+运输到冰箱+
         </span>
<span style="font-size:24px;line-height:150%;font-family:微软雅黑,sans-serif;color:red;">
          信息成本
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          信息成本指的是什么呢。我们看回中学课本。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          “士农工商”，其中商人提供，
          <span style="text-decoration:underline;">
           将货物由甲地运输到乙地，低买高卖等功能。
          </span>
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          其中，“运输”功能，逐渐被社会认可。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          “卡车司机”渐渐被认为一种
         </span>
<span style="font-size:16px;line-height:150%;font-family:宋体;">
          正当职业
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          。归类为“工人”的一体。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          运输是制造业不可或缺的一段环节。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          但是“低买高卖”依然不能被人理解。甚至连“交易平台”，也不能被人理解。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          在一些计划经济官僚的眼中，“集市”就是一个
         </span>
<span style="font-size:21px;line-height:150%;font-family:楷体_GB2312;">
          点
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          ，只要你把牛奶拉到了秦皇岛，就自然能卖得出去。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          这样“朴素”的观点。在前工业化时代是可行的。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          “重工业”绝大多数是大宗交易。例如煤炭，木材，焦炭，铁矿石。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          你如果去过东北的林场，那真的是“堆积如山”。脸盆粗的滚木，一百捆堆成一座小山。垛垛延伸到视线的尽头。
         </span>
<span style="font-family: 楷体_GB2312;font-size: 16px;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>

</p>
<p style="line-height:150%;">
<span style="font-family: 楷体_GB2312;font-size: 16px;">

         </span>

<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          闭着眼睛卖好了。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          可是现代“工商业”社会。根本不是那么一回事。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          最典型的，例如看“京东”。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          京东号称有超过300000个供应商，2500W活跃个人客户。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          每一分钟，都有无数女人在京东上，东逛西逛，乱买东西。
