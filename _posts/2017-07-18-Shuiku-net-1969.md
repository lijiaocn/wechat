<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          他们仅仅是买了一个金融产品，“
          <span style="color:blue">
           IF
          </span>
<span style="color:blue">
           金融崩溃 THEN 可以赚钱
          </span>
          ”而已。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          举个例子，如果你预期房价即将崩盘，则你应该去买“搬家公司”股票。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          因为楼市崩盘，肯定有很多人丧失住所。被扫地出门。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          因而需要搬家。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          你提早买入“搬家公司”股票，等金融风暴。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          自然赚得盆满钵满。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          但是你持有“搬家公司”股票，会导致楼市崩盘么。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          绝对不会。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          这个在逻辑上，属于“关联关系”，而不是“因果关系”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          老人一定有白头发。白头发染黑，你并不会变年轻。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          逻辑是一门大学问。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          其次第二个观点，“大空头”并不意味着看空美国楼市。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          九年了，如果我们平心静气，抹去一切谎言和喧嚣的话。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;color:blue">
          “美国楼市真的跌过么”？
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          随着通货膨胀，楼市每年都有一点的涨幅。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          美国，加拿大，澳大利亚，英国，新西兰，无一不是这样的规律。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          站在2017年回头去看，所谓“2008大跌”只不过长线K图上的一个小波澜。
         </span>
</p>

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          如图，美国楼市最糟糕的时候，也不过是局部地区的一些小麻烦。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          真正WASP贵族精英居住的地区，你过去让人打折，人家正眼都不瞧你。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          “买得起买，买不起滚”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          “大空头”团队发现了算法上的一些bug，真正崩盘的，是衍生证券产品的一些价格。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="line-height: 150%; font-family: 微软雅黑, sans-serif; color: red; font-size: 18px;">
          要指望楼市崩盘，靠CDS显然也不现实。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          三）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          楼市期货
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          真的要真金白银，看空做空，恐怕只能实打实的搞“楼市期货”了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          好比石油期货，橡胶期货，期货和现货是联系在一起的。
         </span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;color:blue">
            期货市场崩盘了，现货市场一定崩盘。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;color:blue">
            期货市场看对了走势，做空就可以赚钱。
           </span>
</p>
</li>
</ul>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          象香港这样以金融，地产立足的亚太都会城市。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          早在1980年代，就开始搞“恒生指数期货”了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          而一直到今天，始终没有“地产指数期货”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          你纵观郑州商品期货市场，芝加哥商品期货市场。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          全世界石油，煤炭期货很多，但是“钢铁”期货很少。为什么。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          因为铜只有一种，石油有二种，煤炭有五六种，钢铁有几十种，而房产有几千种！
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          搞“房价期货”，假设以2017.07.01上海房价为准，起价60000元/m。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          期货交易一个月，双方互有买卖，互有拉锯。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          到了月终结算的时候，收盘价61000元/m。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          如果你对于这个价格不满意，你就要进行“实物交割”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          假设我们交易的单位是“平米”吧。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:13px;line-height:150%;font-family:楷体_GB2312">
          （不是房产专业那个‘平米’）
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          你说你交割内环线1平米。
         </span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
            老破小和汤臣一品，都算1平米么。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
            楼层户型朝向，怎么计算。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
            别墅怎么算
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
            中环边的房子，可以交割么。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
            外环外的房子，怎么折算。对于折算比例不认可，怎么协调处理。
           </span>
</p>
</li>
</ul>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          同样道理，有人实物交割，就有人实物提取。
         </span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
            提取哪一套实物，如何估价。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
            二个人同时看上，如何分先后。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
            手里只有5平米持仓，如何整取。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
            房屋维修养护，如何出租。
           </span>
</p>
</li>
</ul>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          房产这种千奇百怪的东西，怎么能做到整齐化标准期货呢。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          四）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          房产期货的设计
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          这个问题的答案是：
         </span>
<span style="font-size:21px;line-height:150%;font-family:楷体_GB2312;color:blue">
          可以做
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          天底下没有二块一模一样的煤炭，没有二枝一模一样的小麦。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          可是我们照样有煤炭期货，照样有小麦期货。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          这其中的原理是，“忍受轻微损失”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          好比动力煤，因为其内含能量的不同，分为很多个档次。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          但在郑州交易所就统一为TC。如果成色不足，按大卡打折计算。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          美国和加拿大生产的小麦，质量也有好有坏。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          芝加哥交易所统一以“蒲式耳”作计量单位。谷物质量相应换算。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          在整个期货世界，交易量和最终结算的量，超过100:1
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          因此只要游戏规则透明，结算反而是不重要的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          昨天我想了一夜，对于“平米期货”的设计。可以适当调整。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          基于完全市场化下的某一些大数据算法，可以获得非常近似，几乎所有人都接受的折扣计算方式。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          如果真心想做这事，“平米期货”还是开得出来的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          具体的细节，我就不展开了。太琐碎技术化。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          因为俺摩拳擦掌，马上要写最后一段话了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          最后一段话，显然是：
         </span>
<span style="font-size:32px;line-height:150%;font-family:楷体_GB2312;color:red">
          以上全错
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          五）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          对赌
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          我们讲了好几种通俗的“做空房产”方式。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          其中二种是不可行的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          一种是勉强可行的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          其实我们想说的是，“做空房价”这种事，
          <span style="color:blue">
           从理论上就是不可行的。
          </span>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          不是技术上的事，是理论不可行。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          不涉及技术细节设计，理论就说不通。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<blockquote>
<p style="line-height:150%">
<span style="font-size:35px;line-height:150%;font-family:楷体_GB2312;color:red">
           做空的核心是对赌！
          </span>
</p>
</blockquote>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          金融的核心，是风险。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          你看好一项资产，有人看涨，有人看跌。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          傻空摩拳擦掌，兴致勃勃，
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
<span style="color:blue">
           寿星公上吊，找了半天找到一根绳子。
          </span>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          他信心满满，想从“对赌”中赚取二百万。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          可是
         </span>
<span style="font-size:21px;line-height:150%;font-family:楷体_GB2312;color:red">
          任何工具，都是一柄双刃剑。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          你看错了走势怎么办。房价并没有如预料般的下跌，反而大涨了怎么办。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          你辛辛苦苦开了一个期货户头，却不料是催命符。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          各位，房子上涨200W元，是非常非常快的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          你“看空”对赌，
          <span style="color:blue">
           承担风险敞口，就意味着损失。
          </span>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          一旦房价三四个月就涨了贰佰万，
         </span>
<span style="font-size: 27px; line-height: 150%; font-family: 微软雅黑, sans-serif; color: red; background-color: rgb(255, 251, 0);">
          你拿什么来输。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          傻空有200W么。你有能力拿出200W来赌么。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
          拿不出就别瞎BB了，进场的资格都没有。
         </span>
</p>

<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312">
