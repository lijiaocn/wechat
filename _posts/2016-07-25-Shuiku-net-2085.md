<span style="font-size:16px;line-height:150%;font-family:楷体">
          一）借银行
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          借款融资主要有三种渠道：信用卡，银行，私人借贷。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          其中；
         </span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:#365F91">
            信用卡，提供了一个初期虚空借力的方式
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:#365F91">
            银行贷款，90%的主流
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:#365F91">
            私人借贷，抹平波峰
           </span>
</p>
</li>
</ul>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          很多人对于银行贷款的看法，十分浅陋。还停留在“售楼处首付三成，贷款七成”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          甚至还有很多人嫌图利息，不肯贷款。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          以目前的上海市场而论，售楼处平均卖掉100亿房产，贷款22亿左右。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          对于《银行贷款》这一卷，当然不是讨论“贷不贷”，贷几成。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          事实上，我们的回答极为清晰，二个字：“
         </span>
<span style="font-size:29px;line-height:150%;font-family:楷体;color:red">
          尽贷
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          ”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          尽贷，有多少贷多少。对于目前4.9%的利率，有多少贷多少，全力往上限贷款。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          那么，“贷款卷”讲的是什么呢。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          讲的是“有成本下的贷款”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          好比普通的贷款是4.9%，但是他还要收“手续费”。还要收20W元好处费。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          请问你贷不贷。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在这样的情况下，你贷款的“实际成本”就会上升。真实利率不是4.9%，而有可能是6%，8%，10%
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          整个贷款卷，其实就是一篇又一篇的
         </span>
<span style="font-size:21px;line-height:150%;font-family:楷体;color:red">
          精算
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          。计算在各种情况下，你的
         </span>
<span style="font-size:21px;line-height:150%;font-family:楷体;color:blue">
          真实利率
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          是多少。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          然后再由你来决定要不要做。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

</p>

<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">

<span style="font-size:16px;line-height:150%;font-family:楷体">

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在《
          <a data_ue_src="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=207495196&amp;idx=1&amp;sn=4825572100d53270086233f8719760c4&amp;scene=21#wechat_redirect" href="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=207495196&amp;idx=1&amp;sn=4825572100d53270086233f8719760c4&amp;scene=21#wechat_redirect" target="_blank">
           回报率的真相
          </a>
          》一文中，我们说到:“富人其实没有那么富，穷人其实没有那么穷”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          富人的现金资产，可能是穷人的100倍。但他的实际实力，并没有100倍。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为“资源”是一个全方位的概念。
         </span>
<span style="font-size: 16px; line-height: 150%; font-family: 仿宋; color: rgb(122, 79, 214);">
          穷人之所以穷，是因为穷人的资源没有“调度和整合”起来。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size: 16px; line-height: 150%; font-family: 仿宋; color: rgb(122, 79, 214);">
          没有将所有的力量拼拼凑凑起来，拧成一股拳头。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你如果看日本的极道文学，有一股精神是贯穿始终的；
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “
         </span>
<span style="font-size:21px;line-height:150%;font-family:黑体;color:blue">
          将所有的资源整合，将所有的潜力挖掘到极致
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          ”
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          燃烧吧，小宇宙！
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          六）社交的意义
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          有些事，看似是没意义的。但其实又是有意义的。譬如说“社交”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          社交有什么用。你在马路上逮一个屌丝，捏住他的脖子一通乱晃。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “七天之内必须凑齐200W，否则错过这个店”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          屌丝哭丧着脸，“大爷呀，你就算是杀了我。也拿不到200W”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          为什么。因为如果你是那种；
         </span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            一下班就回家
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            公司里只有仇人，没有朋友。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            同乡同学从来不联系，不热络形同陌人。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            不参加网路社团，没有户外爱好，整天宅在家里面。
           </span>
</p>
</li>
</ul>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          则你的各项“指标值”。是非常短的。类似于下图；
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
<img data-ratio="0.6564748201438849" data-s="300,640" data-src="" data-type="jpeg" data-w="" src="{{ '/img/Ok4hZ0tV6r4hKYkcScOmzVmI46Lh3FAN3C06h0YRKEGOuBOTiavvQn9n9kCAwGzic5wB9Ywemxibib7gGiajYgiaC4Ug.jpeg' | prepend: site.img | replace: '//','/' }}"/>
<br/>
</span>
</p>
<p style="line-height:150%">
<span style="font-family: 楷体; line-height: 150%;">
          很多屌丝你让他们“凑钱”。他们是真的凑不出钱。
         </span>
<br/>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为他们不仅仅是“弱”。他们是“全方位弱”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          不仅硬实力家底子穷，父母农民。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而且软实力情商，能动力，个人社交也做得一塌糊涂。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          他们是真的凑不出钱来，哪怕你把“融资利率”提高到10%，12%，15%，每一格也就多凑几万。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而一个富有的“人脉”是什么样子的呢。
         </span>
</p>
<p style="line-height:150%">
<span style="font-family: 楷体; line-height: 150%;">
<img data-ratio="0.6564748201438849" data-s="300,640" data-src="" data-type="png" data-w="" src="{{ '/img/Ok4hZ0tV6r4hKYkcScOmzVmI46Lh3FANUWiagK3Koia7kvAfWZq14icicU41icv2aCfxEjGzxQ4Mp68kMqzGXiaiabaFA.png' | prepend: site.img | replace: '//','/' }}"/>
<br/>
</span>
</p>
<p style="line-height:150%">
<span style="font-family: 楷体; line-height: 150%;">
          图中的红色柱线，意味着“人脉”加成。
         </span>
<br/>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          每一档容量都“接了”长长的一段。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          当你需要融资时，登高一呼，你就能从同学手中借到NN张信用卡。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你想用长钱，男同学们会把自己的私房钱8%放在你这里一年。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你需要调度大笔资金，12%能聚拢几百万。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          一个人花许多时间在“务虚”上面，构建他在社会关系网中的位置。结识更多的人，小心翼翼维护自己
          <span style="color:#984806">
           完美的声誉
          </span>
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:blue">
          这一切，都是在增强你的实力
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          屌丝们平时不努力，事到临头说全世界都欠他的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          呸，别人努力时，你TMD瞎了看不见。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          所谓“人脉”，平日时并不需要动用。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但是当你需要帮助时，“千金散去复归来”。你就可以在短时间内调度大量资金。不论是给你抢笋盘，还是应付突发不可测灾难。都非常地有帮助。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          上几节“信用卡卷”写完之后，有一些清高人士说他们学不会，也不屑于学。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          其实哪怕你永远用不到，你也应该申请一二百万卡额度放着。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          闲着也是闲着。办卡又不伤身。只要不开卡，对你的信用贷就没有大影响。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而到了万一你有危难的时候，宝藏解封，顿时就是源源不断的武器库。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          从03年开始时，我不断地听到有傻空哭诉。怎么可以贷款买房子。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “你怎么不想想失业的风险，万一生大病住医院”。
         </span>
</p>
<p style="line-height:150%">
<br/>
</p>
<p style="line-height:150%">
<span style="line-height: 150%; font-family: 楷体;">
          这些玻璃心都是极脆弱的。我唯一奇怪的是，
         </span>
<span style="font-size: 19px; line-height: 150%; font-family: 楷体; color: red;">
          那些口口声声说“失业医疗费”的人，你怎么不去办几张信用卡？
         </span>
<br/>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
