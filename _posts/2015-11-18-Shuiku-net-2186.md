<span style="font-size:16px;line-height:150%;font-family:楷体">
          一）疯魔流
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          疯魔流是什么，疯魔流就是你的“行动点数”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          玩过《三国志》游戏的朋友都知道，行动点数是什么。好比你这个月的“总行动力”是37点，灌溉农田是4点，训练士兵是5点，挖掘人才是6点，锻造武器是10点。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这样几个指令一加，你这37点就用完了。每个月只能灌溉几块农田。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          所谓“行动点数”，就是指你的主观能动能力。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          好了，现在如果给你74点的行动力，请问会怎样。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          答案是刘备的势力，会像打了鸡血一样急剧膨胀，攻占54座城池通关。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “疯魔流”指的是一种精神。一种每天有
         </span>
<span style="font-size:21px;line-height:150%;font-family:楷体;color:red">
          48
         </span>
<span style="font-size:21px;line-height:150%;font-family:楷体;color:red">
          小时
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          的精神。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          很喜欢负能量的一句名言，“绝大多数人努力程度之低，根本谈不上拼天赋”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          对于我们这个社会上99%的人，命运中的所有奋斗，大概从“高考放榜”那一天之后就已经结束。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          从此之后，他们再也不学习，再也不进步。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          绝大多数人口，尤其是北方人，东北人，他们的理想就是进了国企，进了铁饭碗，然后在一个“年复一年”不断重复的工作中，永远也不要动脑。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          譬如会计，譬如HR培训师，譬如R&amp;R
          <a data-ratio="0" data-w="15" name="_ftnref1" style="font-size: 10px; text-decoration: underline;" title="">
</a>
<span style="font-size: 10px; line-height: 150%; font-family: 楷体;">
           [1]
          </span>
          ，譬如机关里的资料审核员。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果你和他们讲刘强东挖空心思卖光盘，求伯君几次创业换方向，褚时健八十岁卖橙子-------他们是不能理解的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在他们的脑颅中，大脑已经生锈，血管已经堵塞，18岁之后激情就再没有燃烧过。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

</p>

<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">

</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          二）神啊，再赐我一点时间
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          对于真正想进步的人，时间是永远不够的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          不知道多少人还有这种体会，当你打第一份工时，向苍天呼喊说“神啊，请再多给我一点时间”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          一天24个小时是绝对不够的。尤其是吃饭睡觉又占去了一大块。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          当一个新的领域，新的天地向你打开。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          当你做了第一笔单子赚了20W，一个全新的赚钱模式向你开放。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          当你只要去做，你能赚N个20W。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这时候，时间是绝对不够的，每一天最好能有48小时。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你只觉得浑身的血液都在燃烧，每一天都在奔跑，新的知识新的养分新的商机新的钞票，尽情地大口吞噬。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          所以“疯魔流”还不如说是一个鸡血流派。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          只要你开始对自己高要求，发光发亮。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          别人二年炒一套房子，你一年可以做4套单子。能拼能冲能干，“心力”爆棚。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          三）疯魔流之邪途
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          疯魔流有一个邪路，我们希望各位不要算在水库头上。
         </span>
</p>
<p style="line-height:150%">
<span style="font-family: 楷体; line-height: 150%;">
<br/>
</span>
</p>
<p style="line-height:150%">
<span style="font-family: 楷体; line-height: 150%;">
          譬如深圳有一个流派，可以称之为“邪恶疯魔流”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这个流派的做法是什么呢，他们基于一个原理：“既然炒房是赚钱的，我们就拼命炒”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在之后的文献中，我们会说到，炒楼这一行，平均年化回报24%大致还是有的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但是在“邪恶疯魔流”手中呢，就变成了“只要36%以下的资金我都敢借”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          只要月息3%以下资金都敢借。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果你想扩大规模，别人买1套房子，你买2套房子，未来买4套8套房子。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          则你需要的“本金”也增加一倍。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “邪恶疯魔流”就是
          <span style="text-decoration:underline;">
           高利贷借本金去炒楼
          </span>
          的一种流派。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          最有趣的是，从理论上，这个流派还是完全正确的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为过去十五年，乃至未来二三年的实际情况，炒楼的确是有24%的回报。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          考虑到利息加权，你最终边际利率借到36%，你仍然是有钱赚的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          也就是在“大牛市”中；
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:red">
          这种不惜一切，不顾一切，拼命借所有能借到的钱，亡命一样地扩张模式，反而是最赚钱的
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          。是所有流派中回报率最高的一种。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但是，做人做事做生意，毕竟不可以玩命。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “水库论坛”的创始人基本都是
          <span style="text-decoration:underline;">
           上海
          </span>
          多军。沪派江浙，和广东人的风格还是很大不同的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          那什么样的几率可以博呢。按照期货标准交易守则“赢三博一”，也就是胜率：负率是3:1的时候，才可以下单。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          所以
          <span style="color:blue">
           上海多军能接受的加权资金利率是8%
          </span>
          ，上海人始终是很保守的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          邪恶疯魔流”指的是一种完全不顾成本，不顾风险，用命去博，疯狂扩张的流派。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在大牛市中，这个流派赚得最多。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但我们还是希望和他们划清界限。真的有一天，你们别说是“水库传人”出来的。您还是说金朝阳学生吧。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          四）善良的疯魔流
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我们所鼓励，所提倡的疯魔流是什么呢。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          指的是“不额外增加风险的情况下，通过增加
          <span style="text-decoration:underline;">
           勤奋
          </span>
          ，而获得更多回报”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这个世界，绝大多数人是不勤奋的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          哪怕你把炒楼秘籍全部都告诉他们，许多人可能也懒洋洋只有10点行动力。二年才买一套，而且功课还做得不足。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而如果你足够勤奋，就可以后生可畏，后来居上。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但是我们不提倡因为疯魔流，而增加额外风险。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          不提倡高利贷融资。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          华北，华东，华南，差距真远。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
<br/>
</span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          五）转租流
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          当使用到了“法租+药单+周转+疯魔”层层加持，一步步把投资回报逼上去之后。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          请问，还有没有办法，进一步提高利润？
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          答案居然还是有的：
          <span style="font-size: 16px; line-height: 150%; font-family: 楷体; color: rgb(255, 0, 0);">
           轻资产
          </span>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          轻资产是什么，轻资产大概是整个21cn美国资本市场最热门的话题。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “买房子收租子”，会计角度你的利润就是房租。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          可是这个房租，回报率是很低的。你通常要买到500W的房产，才能获得8000元的月租。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          可是，如果现在有人告诉你，你只要20W元的资产，同样也可以获得8000元/月的房租。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          请问，你这个投资回报率怎么算？
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          投资人会不会要发疯，资本市场要不要发疯，美国的金融市场要不要发疯。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          整个1990
         </span>
<span style="font-size:16px;line-height:150%">
          ~
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          2015
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          年，基本就是美国的
          <span style="color:red">
           工商业界
          </span>
          卖资产的故事。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在华尔街，投行的前辈们有一句口耳相传的经验：“绝对不要持有任何Factory”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          华尔街对所有的“重资产”避而远之，无论是机器，煤矿，铁路，轮船，卫星，摩天大厦，电力设施，海底光缆。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          他们只希望你签二纸薄薄的合同，通过金融或者贸易，把所有的利润全都拿到手。华尔街喜欢虚拟资产。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          六）二房东
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          五年前我遇见一个人，我问他有几套房子在出租。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “756套”，俺不由得吃了一惊。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “为了这事，我专门请了3个人全职收房租，8个人打扫卫生，2个修马桶，还有2个人专职负责交水电费…………总计一共是养了一个22人的团队”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “…………”
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<img data-ratio="0.8003952569169961" data-s="300,640" data-src="" data-type="jpeg" data-w="" src="{{ '/img/Ok4hZ0tV6r419fAAgZyze2Et3QBOiaoFKAmsAD4khBJjWqyQ5XSXQBcfkVcKH7ODeP0ZRYelDdPBiasDwQojdXwQ.jpeg' | prepend: site.img | replace: '//','/' }}"/>
<br/>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但其实呢，他的资产并没有你想象中那么高。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为他虽然有756套收租物业，但他是“二房东”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          规模做得特别大的水底怪兽。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果你单纯地要说“业务回报”的话，则“轻资产”模式才是真正做到了尽头。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          二房东的资金投入，相对于大房东来说，不到1/10。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而在租金的分配上，一个装饰良好，能抓住客户欢心的二房东，完全可以将租金溢价1：1
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这样一个模式，如果你做成App，放到移动互联网概念上。网上订房。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          有良好的赢利模型，无穷无尽的市场深度，强悍之极的财务数据，高得吓死人的增长速度…………
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          哇塞，秒杀掉市场上90%的O2O，应该不成问题。我们是一家赢利公司，而且一开始就赢利的极高速增长轻资产公司。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          估值不估个三十亿，五十亿，都不好意思出去见人。你投个1000W，只能给你A轮1%股份。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          可惜的是，证监会规定房地产类公司不许上市。洗洗睡吧。
         </span>
</p>

<span style="font-size:16px;line-height:150%;font-family:楷体">
