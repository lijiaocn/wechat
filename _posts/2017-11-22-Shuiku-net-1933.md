</p>
</blockquote>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          学历并不是越高越好。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          最笨的人才去读研究生。考研是低智商，低情商的表现。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          对于内陆省份来说，上不了211，野鸡大学的学费，回报就是负数。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          还不如考个Blue Fly。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          对于京沪大城市来说，本科之后的教育，也是个负数。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          决定人生的是高考。第一学历才是学历。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          “教育”这种事，本质是一种投资学选择。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          在研究生“录取率”如此高的今天，绝大多数的人，并不是因为“考不上”而不读研究生。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          他们纯粹是因为“不划算”而不读研究生。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          研究生本身，也并不意味着更高的情操，或者更高的审美品位。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          譬如说，你每天看着200对果蝇交配，连续看上二年。你觉得这种事很有审美么。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<blockquote>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
           但是中产阶级并不明白这个道理。
          </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
           在目前的“Mid-Low Class”中，对“教育”的投资，几乎是
          </span>
<span style="font-size:21px;line-height:150%;font-family:微软雅黑,sans-serif;color:red;">
           压倒性，一面倒，不计成本的
          </span>
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
           。
          </span>
</p>
</blockquote>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          我们知道，经济规律这种事，你可以不理睬它。但是它不会不理你。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          不计成本，不计回报，
         </span>
<span style="font-size:19px;line-height:150%;font-family:楷体_GB2312;color:blue;">
          无限制的教育投资，恶果显现。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          我们看到，一些Mid-Low Class的父母，本身可能仅仅是一个营业员，一位卡车司机。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          但是他们会倾注全部的资源，省吃俭用，把全部的家财，堆积在孩子身上学钢琴。并期冀孩子可以成才。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          我们并不是说，学“八国外语”不好。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          我们指的是，“经济学”才是高等科学。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          凡事都要讲“费效比”。你大把投资，牺牲掉父母所有的人生和幸福。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          能成才么，不能。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          能赚钱么，不能。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          能快乐么，不能。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          最终的结果，是一个36岁嫁不出去的生物女博士，绷着马脸，在五星级大酒店大堂里弹琴卖艺维持生计。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-family: 楷体_GB2312;font-size: 16px;">
          您是不配作为套房客人的，你只配当个服务生。

<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          三）移民投资
         </span>

<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          中产阶级手里有了100W元。不满足于传统的股票，债券投资。想要高人一等，出人头地。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          除了砸在孩子身上，进行硕博教育。还能干什么。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          还能出国。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<blockquote>
<p style="line-height:150%;">
<span style="font-size:19px;line-height:150%;font-family:楷体_GB2312;color:blue;">
           我真心不明白，一些40岁事业有成，单位实权骨干，为什么要急吼吼出国！
          </span>
</p>
</blockquote>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          中国有几件“政治正确”的事。老百姓习以为常，路径依赖。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          自以为天经地义，实则荒谬绝伦。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;color:blue;">
          教育是一项，出国又是一项。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">

<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">

<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
            哪些国家，用何种方式，什么渠道出国。
           </span>

<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          对于这些问题，中产阶级一无所知。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          你如果和内陆省份一些家庭聊过，他们的思想，基本是“
          <span style="color:red;">
           为出国而出国
          </span>
          ”。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          仿佛你只要一抵达美洲大陆，白银珠宝都会滚滚而落，直接跳出“游戏通关”的字幕。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          而瞎眼罔顾美国欧洲也有大量穷人，黑人老墨，生活水准远远不如京沪的事实。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          中产阶级，是一个很特色的阶层。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          严格来说，他们并不是上层阶级。而仅仅是“模仿”上流社会。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          而这个模仿，
          <span style="color:blue;">
           又不是实时的。而是时滞很严重的
          </span>
          。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          1980年代，国门刚刚打开，欧美港澳台收入20000元/月，中国人收入80元/月。差异极为巨大狂野。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          当时的京沪粤，沿海大城市，人们惊讶地发现，去日本打黑工二年，回来就是农村首富。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          所以80年代形成了巨大的“出国潮”，并因此被内陆省份模仿和牢记。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          但是，时移世易，到了2017年，京沪的上层生活水准早已远远超过欧美中产。“移民”对于A8阶层来说，毫无任何吸引力。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          三里屯的老外，都变成White Trash了。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          因为啤酒涨价，最近开始改喝可乐。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
<br/>
</span>
</p>
<p>

</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          但是，中产阶级和上流社会的区别是，中产阶级只具备“模仿”能力，中产阶级是不具备“思考”能力的。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          当上海A8早已不屑欧美穷鬼时，内陆的中产阶级，依然飞蛾扑火一般奋勇出国。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          出国不是手段，出国就是目的。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          出国人云亦云，不算收益成本。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          盲从的下场，必然又是一次
          <span style="color:blue;">
           排队枪毙
          </span>
          。
         </span>

<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          一个中产家庭，一旦出国，整个家庭就算毁了。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          从零做起，慢速轨道，回天无力。中产迅速返贫。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          四）艺术投资
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          第三个陷阱地雷，是艺术。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          尤其是画画，
          <span style="color:red;">
           女孩子学画画，一定要坚定拒绝。
          </span>
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          我们看韩剧，
          <a href="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=210009763&amp;idx=1&amp;sn=4c6655048bfe023db1720a9c5fd9adab&amp;scene=21#wechat_redirect" target="_blank">
           韩剧里
          </a>
          ，有一个非常典型的范例。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          富家贵女，例如社长的妹妹，嫁不出去的姑妈，刁蛮的大小姐，富商千金，无一例外，她们的专业是“美术系”。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          这些贵妇，贵小姐们，并不需要上班打表，维持生计。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          可是她们在“专业”系科选择上，无一例外选择了绘画，更远远超过第二名的音乐。为什么。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          因为绘画，
          <span style="color:red;">
           是一门“不用竞争”的学科
          </span>
          。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          一旦你选定了画家作为终身职业，你就再也不用面临激烈的竞争，残酷的排位赛制考试。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          当你女儿，当你的小孩，16岁时向你提出“想学画画”。此时，你一定要坚决拒绝。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          当她说出这句话时，她的潜台词其实是，“
         </span>
<span style="font-size:21px;line-height:150%;font-family:楷体_GB2312;color:blue;">
          我已厌倦了激烈的竞争，我已害怕努力和奋斗，我宁愿做一个懦夫躲起来藏起来
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          ”。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          于是，贵族小姐，可以堂而皇之的找一个画室躲起来，可以IB。反正可以用父母的钱，衣食无忧。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          她可以毫无压力地工作，花几个月的时间画一幅画。别人甚至还不能评论“艺术”的工资。尽管她们一辈子也卖不出去一幅画。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          真正的绘工大师，是极其辛苦的，也是极度清贫的。日夜勤勉。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          那笔锋，刀锋，每一划都是灵魂之作。用生命在绘画。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          而贵妇们不是。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          她们是如此之懒，懒到连“音乐”都不愿意选科。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          你要当众演奏一段小提琴的话，至少你也是刻苦训练锻炼，反复练习的。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          琴弦的轻微桎梏和生疏，观众是轻易可以听出来的。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          无论你是小有身家的企业家，还是自恃清高的知识分子。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          当你的女儿，向你申请“学画画”时，一定要坚决拒绝。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          家族如此富裕，再多养一个人口，本身不算很大的负担。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          但真正的麻烦是，“躲避竞争”是一个潘多拉魔盒。如鸦片毒瘾，千万不能沾。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          在此后她的人生中，会有无数无数杜十娘问题，象连环炸弹一样，不停爆发出来。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          五）结语
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">

<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          前二天我见了一个温州行业会长，问他温州近期经济怎样。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          他说老一辈的企业家还都活着，每一个都是人精。聪明狡猾，诡计多端，要收割老狐狸们实在不易。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          然并卵，哪怕你再牛强，421的规律，总是避免不了的。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          家族财富，
          <span style="color:#E36C0A;">
           四个再强大的局长处长，最终只留下了一个傻白甜
          </span>
          ，女继承人。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-family: 楷体_GB2312;font-size: 16px;">
<br/>
</span>
</p>
<p style="line-height:150%;">
<span style="font-family: 楷体_GB2312;font-size: 16px;">
<br/>
</span>
</p>
<p style="line-height:150%;">
<span style="font-family: 楷体_GB2312;font-size: 16px;">
          我再问他，那么继承人培养得如何。“出国留学投资读书艺术科目”效果如何。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          他苦笑着说，出去了二十五个人。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          其中二十四个人，辛勤读书，学成归国，高科文凭。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          只有一个学渣，在美国整天跑车把妹，考试参考“天才枪手”。混了一个文凭。
         </span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
</span>
</p>
<p style="line-height:150%;">
<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
          然后你猜，二代中有能力继承家业的，成才了几个？
         </span>

<span style="font-size:16px;line-height:150%;font-family:楷体_GB2312;">
