<ul class="list-paddingleft-2" style="list-style-type: disc;">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          别人在小密圈问我，“能否去大城市配置资产”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          高宗搞出来的“商品房”。商品房作为一种商品，而不是组织分房。其最基本的本意：
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:微软雅黑,sans-serif;color:red">
          商品，拿钱就可以买。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          也就是说，
          <span style="color:blue">
           市场经济，只认钞票。
          </span>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          可是搞到今天，已经完全变了味了。大城市的土著，就是特权群体。土著有房票可以买。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          外地来的土财，单身没结婚还不能买。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          仅一个大城市的户籍身份，就可以价值几十万上百万。
         </span>
</p>
<p style="line-height:150%">
<span style="font-family: 楷体; font-size: 16px;">
</span>
<br/>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          对于房票+钞票这个组合，还可以加上十几个细化参数。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<span style="font-size:16px;line-height:150%;font-family:楷体">

<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

</span>

<p style="line-height:150%">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

</p>
<p style="line-height:150%">

         </span>

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          北京的房地产市场，他是“点状”的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          好比一块大饼，零零碎碎洒了芝麻。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          有芝麻的地方，就是平地蹿起一个个100000+的楼盘。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          可是周边大块空地呢。对不起，没你的份。北京是不存在“地段”的。北京纯粹是一个“点”状市场。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          北京地产逻辑，简直可以说“土豪驱动”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          土豪看中某一块地方，霄云路八号，直接几千万砸下去。买房也就是买个大玩具。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          不需要考虑横比，竖比，性价比。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size: 16px; line-height: 150%; font-family: 楷体; color: rgb(2, 30, 170);">
          北京有钱人，不是身家几千万的人群，而是零用钱几千万的人群。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          某一个地块，一旦崛起一家互联网风口期公司。顿时鸡犬升天。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          股票，PE估值千百倍的增长。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          公司一夜之间多出来几个营的千万富翁。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          把附近楼盘全部买空。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而过了几年，风口过去。一切又回归平静。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          西二旗的互联网公司，渐渐竞争不过望京。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          整个土地地段估值也瞬间转移。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          所以在北京，“地产”是非常非常难做的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          它根本没有“地段”的概念。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你不知道土豪下一秒会青睐哪个位置。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          也不知道互联网哪家公司会崛起。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          同样道理。深圳整个城市，基本是“带状”分布。政府一路向西，开发新区。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          前二天去看了华侨城，心都凉了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          深圳最早的地标性顶级豪宅，整个城市的名片标杆。富豪云集，万金买邻之地。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          20年来，保值了没有？
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          现在价格才8W不到。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          深圳因为是“带状”城市，政府一路往西。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          从最初的罗湖，香蜜湖，南山，到深圳湾，前海，后海，宝安，宝中。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          政府永远在开发新区。
         </span>
</p>
<p style="line-height:150%">

<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">

<p>

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在《
          <a data_ue_src="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=403391036&amp;idx=1&amp;sn=2416ee9f1731daca3905e605fbfbe55a&amp;scene=21#wechat_redirect" href="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=403391036&amp;idx=1&amp;sn=2416ee9f1731daca3905e605fbfbe55a&amp;scene=21#wechat_redirect" target="_blank">
           炒楼业黄金时代
          </a>
          》#930及前后的二三篇文章中，我们详细说道了：
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “
         </span>
<span style="font-size:16px;line-height:150%;font-family:微软雅黑,sans-serif;color:red">
          宏观调控之后，炒家利润率急剧升高
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          ”的结论。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          美国“禁酒令”颁布之后，才是酒贩子赚钱的黄金时代。卖卖沐浴露之类的。
          <a name="_ftnref2" style="font-size: 10px; text-decoration: underline;" title="">
<span style="line-height: 150%; font-family: 楷体; font-size: 10px;">
            [2]

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          经济是一件很玄妙，很复杂的事情。和大部分人的直观完全不同。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          炒楼从来没有这么赚钱过，尤其是2012
         </span>
<span style="font-size:16px;line-height:150%">
          ~
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          2015卡得最死，最严的时段。最赚钱。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<blockquote>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:仿宋;color:fuchsia">

<span style="font-size:16px;line-height:150%;font-family:仿宋;color:#00B050">

<span style="font-size:16px;line-height:150%;font-family:仿宋;color:fuchsia">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:仿宋;color:fuchsia">
           在广袤无垠的大草原上，只有孤零零几只孤狼。
          </span>

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:仿宋;color:fuchsia">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:仿宋;color:fuchsia">
           青霉素杀死了99.9999%，却让0.0001%独享美食。
          </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
<a name="_ftnref3" style="font-size: 10px; text-decoration: underline;" title="">
<span style="line-height: 150%; font-family: 楷体; font-size: 10px;">
             [3]
            </span>
</a>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:仿宋;color:fuchsia">

<span style="font-size:16px;line-height:150%;font-family:楷体">

<span style="font-size:16px;line-height:150%;font-family:楷体">
          AK47用完了，开始用狙击枪了。

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          世上没有难赚的钱，只有不思进取的人。

<span style="font-size:16px;line-height:150%;font-family:楷体">

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          要说赚钱，北京其实才是最好的市场。哪怕北京杠杆率最低，限购最难，板块最混乱。北京是Hell模式。
