          一）书籍
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          BC700，佛祖时代的经文，被称之为《贝叶经》。是用针刺在贝树叶子上的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          贝叶经极其容易损毁，携带也不方便。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          此后漫长的封建岁月，书籍的主要载体是：羊皮。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          虽然羊不算是很贵的动物，但数百张羊皮依然是很耗钱的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我们想说：“在古代，书籍是非常非常稀少的”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          书籍稀少到怎样一种程度。中世纪欧洲，如果一个人写了一本书。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          那么他就把他的左手小指砍下来。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          用根绳子，挂在书的扉页上面。表示这样的书再也不会有。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          书比人更值钱。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在《临高启明》里讲了一个段子。临高集团想要贿赂海南书院。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          送过去任何礼品，刘大霖都是不会收的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但是临高集团，送的是一套现代印刷机精印的《四书五经》。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          海南书院，虽然也号称书院。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但是以一县之力，甚至连一套齐的《四书五经》都凑不出。书籍昂贵，纸张昂贵。更不要说精装版，便携版了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在《文明》系列游戏中，向来把“印刷术”列为一项非常重要的科技。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          活字印刷，造纸，指南针，火药，被誉为中国古代四大发明。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这里面是非常非常有道理的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “造纸”的出现，使得竹简的成本一下子降了二个零。为大规模的识字和文化传播奠定了基础。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而文字和吏治国家，则是一个辽阔帝国运转的枢石。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          可以说，没有刀笔吏，就没有大型帝国。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果没有“造纸”。中国会重回春秋，以分封制为主。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而“印刷术”的发明，使得人类科技文明传播，再大大向前走了一步。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “羊皮纸”时代，一本书只有一本，仅限于非常重要的宗教和巫术记载。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “造纸”时代，识字属于贵族。“洛阳纸贵”开启了东晋世代门阀。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “印刷术”时代，掀起了报纸大众知识革命和民煮浪潮。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          一般认为1949年之前，中国的“识字率”是1%
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          一直到80年代，我们读书那阵。很多小学生还有记忆，学校里有蜡染工坊，有油纸和穿着工作服的印考卷师傅。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          王选凭借其“汉字激光照排技术”，在改革开放初期获得了巨大的声誉。创立方正集团。史玉柱研制汉卡。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          他们的名字，不该被忘记。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          然后就是“五角丛书”，书籍的价格大幅降低。一直降到今天卖白纸的价格。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          收集一套全本柏杨《资治通鉴》72册，再也不是什么难事。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          甚至在旧书市场地位至高无上的“直版繁体”，前二天也有人送了我一套《笑傲江湖》。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我们唠唠叨叨写了八九百个字，想表达什么意思呢；
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:32px;line-height:150%;font-family:楷体;color:red">
          “以上全错”

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>

<p style="line-height:150%">

<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          有人开玩笑说，“书籍已经变成一种易腐烂商品。其保质期
          <span style="text-decoration:underline;">
           介于牛奶和酸奶
          </span>
          之间”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          Amazon于是想出了一个计划。他们直接和作家签约。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          作家写完Word文件，直接封订在Amazon的橱窗栏里面。
         </span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            没有库存
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            没有保质期
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            没有印刷
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            没有展示位限制
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            不需要听挑剔的，没有品味的，低级庸俗的出版商“买手”苛刻刁难。
            <em>
<span style="color:#7030A0">
              老混蛋不懂真经非说我这书卖不出去不肯给我封推展示位下架鄙视你垃圾。
             </span>
</em>
</span>
</p>
</li>
</ul>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在我看来，Amazon这想法是好的。也极其充满了创意。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          对传统的书商行业，是颠覆性的革命。一点不给实体书店活命。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但我还是要说一句；
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “
         </span>
<span style="font-size:27px;line-height:150%;font-family:楷体;color:red">
          以上全错
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          ”
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
<br/>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          四）书籍时代的消亡
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在我看来，Amazon依然是落后于时代了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          他想出了一个极其有创意的图书分销模式。但是革命不彻底。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<blockquote>
<p style="line-height:150%">
<span style="font-size:19px;line-height:150%;font-family:宋体;color:blue">
           他没有想过，连图书这种信息形式，本身都不存在了。
          </span>
</p>
</blockquote>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          图书是什么。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          图书本身是一连串的章节。譬如《红楼梦》1
         </span>
<span style="font-size:16px;line-height:150%">
          ~
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          80回。每一回大约6000字。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          《红楼梦》排版顺序是，第一回，第二回…………第八十回。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          那么，它简直就是80本薄薄的小册子。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:#984807">
          用订书机，蛮力装订在一起。
         </span>
</p>
<p style="margin-top:10px;line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我们不由得想问，这样的“格式”，合理么？
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          举个例子，yevon_ou在写“水库论坛”微信公众号。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          从第一回《
          <a data_ue_src="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=204836730&amp;idx=1&amp;sn=39bb60c5ae70746e60cca2e78bbab5e4&amp;scene=21#wechat_redirect" href="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=204836730&amp;idx=1&amp;sn=39bb60c5ae70746e60cca2e78bbab5e4&amp;scene=21#wechat_redirect" target="_blank">
           毁三观，竖新生
          </a>
          》#1，到本篇，零零碎碎也有二三百篇了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我们的格式是，每隔几天写一篇。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          每一篇的末尾，有网友的打赏，评论，和点赞。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          作者可以不停和读者互动。读者有时候还会点菜，“什么时候写写月供啊”，“最新加息政策怎么看”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而作者写到一半，可以引用前几篇的文章。直接加个链接，例如《
          <a data_ue_src="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=2651015310&amp;idx=1&amp;sn=8bdd05b874cdf3e792e3bfcf33ba0e5f&amp;chksm=80721e9db705978bd3557c5753fd213c4df4e1ea75392e6a3570dcf503646ae14419fd35d30a&amp;scene=21#wechat_redirect" href="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=2651015310&amp;idx=1&amp;sn=8bdd05b874cdf3e792e3bfcf33ba0e5f&amp;chksm=80721e9db705978bd3557c5753fd213c4df4e1ea75392e6a3570dcf503646ae14419fd35d30a&amp;scene=21#wechat_redirect" target="_blank">
           中产阶级如何保护自己的财富
          </a>
          》
          <span style="line-height: 150%; font-family: 楷体; font-size: 10px;">
           #F730
          </span>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          最绝妙的，是作者还可以“改”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          譬如说，今天看到一篇非常好的文章:《易中天：“达康书记”也是你能叫的？》
          <a name="_ftnref2" style="font-size: 10px; text-decoration: underline;" title="">
<span style="line-height: 150%; font-family: 楷体; font-size: 10px;">
            [2]
           </span>
</a>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我在《
          <a data_ue_src="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=211128132&amp;idx=1&amp;sn=ca586ee1556e5e43c9949624294e49e6&amp;scene=21#wechat_redirect" href="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=211128132&amp;idx=1&amp;sn=ca586ee1556e5e43c9949624294e49e6&amp;scene=21#wechat_redirect" target="_blank">
           论儒·法·禅宗·基督自治与道教的复兴
          </a>
          》
          <sub>
           #F10
          </sub>
          一文中说，“儒教”也被称之为“礼教”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          儒的本质是等级制。而等级制需要确权，需要确定双方的长幼尊卑。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<blockquote>
<p style="line-height:150%">
<span style="line-height: 150%; font-family: 宋体; font-size: 14px; color: rgb(0, 122, 170);">
           二个人见面需要行礼。行礼的方式和言语，其实就是“地位等级”的暗中衡量。
          </span>
</p>
<p style="line-height:150%">
<span style="line-height: 150%; font-family: 宋体; font-size: 14px; color: rgb(0, 122, 170);">
           行礼，决定了二个人以后相处的尊卑服从次序。
          </span>
</p>
</blockquote>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这段话写的时候，苦于没有太好的“
          <span style="color:red">
           称呼行礼
          </span>
          ”参照，行文寡淡无味。今天看见易中天这篇文章，如获至宝。于是俺高高兴兴地把参考“注释”加到了#F10中去。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果我们把“水库论坛”视为一课“炒楼学”教科书的话。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:blue">
          你是买一本书的效果好。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:blue">
          还是关注一个微信公众号的效果好。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          答案毫无疑问是“微信公众号”的效果好。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为微信公众号他是“活”的。可以有互动，有更新，有改良。甚至每一次打开，内容都会有轻微改进不一样。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          有人说，手机拍照，手机录像，自媒体兴起了以后。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          报纸“报导新闻”，就再也没有人看了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为报纸是一种非常非常缓慢的工具。今天晚上发生的事情，最快也要到明天晨报才能发刊。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          朋友圈早就刷屏，刷烂掉了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          同样道理，我对于书籍的怀疑是；
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
<br/>
</span>
</p>
<blockquote>
<p style="line-height:150%">
<span style="line-height: 150%; font-family: 微软雅黑, sans-serif; color: red; font-size: 18px;">
           “作为一种知识组织形式，‘书’是否已经过期”。
          </span>
</p>
</blockquote>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          各位，我们看待问题，一定要看清本源。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          书籍，不过是一种知识学习途径。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          上课，也是一种知识学习途径。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          刷朋友圈，刷知乎，也是一种知识学习途径。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          甚至连“社会大学”，也是一种知识学习途径。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:微软雅黑,sans-serif;color:fuchsia">
          它们并没有什么不同。核心都是
          <span style="text-decoration:underline;">
           学习知识
          </span>
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          目前社会上对“碎片化时代”颇有微词，学术界普遍对碎片学习时间不屑一顾。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这是不对的，屁股决定脑袋。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:#00B050">
          是教书匠为了维持自己的职业特权
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          ，而在进行辩护。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          真正导致“碎片化”形象低下的，是
         </span>
<span style="font-size:21px;line-height:150%;font-family:黑体;color:red">
          没有考试
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          没有考试，则你不能甄别学生质量。罗振宇的每一堂演讲，都必须假设台下都是
         </span>
<span style="font-size:16px;line-height:150%;font-family:仿宋">
          完全没有基础的小白
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          。从而降低了演讲质量。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果能够对学生进行真正意义上的
         </span>
<span style="font-size:19px;line-height:150%;font-family:微软雅黑,sans-serif;color:red">
          严格考试
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          考不通则留级。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          则函授，或者网络大学的效率，
          <span style="color:red">
           是会超过课堂的。
          </span>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          哪怕老师拉个群，让学生在群里自由讨论。老师不时点播指引（小密圈），这样的效率，都好过你在课堂里枯坐45分钟。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          Amazon研究的是“书籍”的分销模式。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而我则是担心，未来连“书籍”都不复存在了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          五）未来的知识
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:blue">
          书籍只不过一种表象，其本质是知识的传播。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          未来知识的传播，不一定要靠“80回装订”成一本红楼梦的方式进行。完全可以VR趟一回。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          所谓书籍，作者写成了一篇篇6000字左右的“章节”，再机械地装订在一起。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size: 16px; line-height: 150%; font-family: 楷体; color: rgb(2, 30, 170);">
          这样的“知识组织”形式，何等的可笑！
         </span>
</p>
<p style="line-height:150%">
<span style="font-size: 16px; line-height: 150%; font-family: 楷体; color: rgb(2, 30, 170);">
          完全是中古时代的遗留物。化石未进化完全的遗骨。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          未来的时代，“知识学习”一定有更多的组织方式。例如微信公众号，订阅推送。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          小密圈，私密辅导答疑。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          然后你再去实战一把。实战中遇到困难和困惑，到群里面来问。大家帮助。难保还能借你点钱。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这样的成长速度，是非常非常快的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          时代在发展，“知识组织”的形式在变革。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          20年之后，我们的下一代，很可能就完全不需要书籍了。或许也完全不需要大学，课堂，校园。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在这整个时代落幕之前，您有限的经费，不如只购买一本书吧。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          也请
          <span style="color:blue">
           多多赠送亲朋好友
          </span>
          。
         </span>
</p>
