<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “假如上海楼市彻底崩溃，你会破产么”。
         </span>

<p style="line-height:150%">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          二）中立
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “假如上海楼市彻底崩溃，你会破产么”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          很多人立刻会沾沾自喜地说：“不会”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果我们看回“流派”的诸多文章，会发现许多流派都宣扬自己“牛熊中立性”。包括药单流，民宿流，周转流…………
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          药单流，他赚的是解药的钱。周转流，他赚的是“凤姐变冰冰”的钱。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这些流派，都是和牛熊市无关的。无论行情好，行情坏，他们照样赚钱。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          周转流”，平均一套单子在手里只持有四个月。这短短时间内，大市再如何波动也涨跌不了多少。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而“民宿流”“老破小流”注重的则是租金，赚的是房租。其实也不关心房价的涨跌。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          牛熊中立性”是一件好事。意味着你避免了房价的大跌。譬如药单流，就号称永不亏损。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          可是，真正的高人，应该想深入一步。假如房价并没有如你“所愿”大跌了，反而是大涨了，你会怎么办？
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<blockquote>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:red">
           风险的另一面剑刃，是收益。
          </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:red">
           没有风险的另一面剑刃，是没有收益。
          </span>
</p>
</blockquote>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          当“八大流派”不约而同地
          <span style="text-decoration:underline;">
           规避
          </span>
          了风险的同时，他们也就失去了收益。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          意思就是，当房价涨五倍，十倍，其实你并没有赚多少。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          三）狂赚
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果你是一个金融金领，你可以赚多少。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          30W
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          ，50W，80W，150W，500W，还是更多…………
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而如果你业余炒股票，你可以赚多少。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          假设500W投在股市里面，一个涨停，顿时就是30W~50W的利润。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这一轮行情下来，很多人已经有了200W的利润，远远地超出他们平均工资。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          哦，是否
         </span>
<span style="font-size:16px;line-height:150%">
          ~
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          大家都别上班了，热热闹闹地去炒股票好么。赚得多啊。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          话也不是这么说，因为“股市有风险”。有可能涨，也有可能跌。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在每季度赚200W的背后，也有可能亏200W。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          那么，你就面临二种模式的选择。
         </span>
</p>
<ol class="list-paddingleft-2" style="list-style-type: upper-alpha;">
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            老老实实上班，赚30W一年
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            炒股票。可能赚200W，也可能亏200W
           </span>
</p>
</li>
</ol>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<blockquote>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:blue">
           朋友们，你们要知道，在这个世界上，赚得最多的是哪一个行业。
          </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:blue">
           赚得最多的，是“担风险”的行业。
          </span>
</p>
</blockquote>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在古代，是杀官造反当皇帝。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在现代，是投靠站队分派系。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在公司里，在官场上，每个人都可以选择，是做“技术性老黄牛”，还是做“政治性站队者”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          其中，技术性老黄牛论资排辈。拿30W一年，既饿不死，也吃不饱。你的一生就这样忙忙碌碌过去。老来时拿三分饱的工资退休。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而“站队”这种事，九死一生。站稳了，青云直上，一步登天。哪怕是在公司里，跟上一个特别旺的老板，你也能连升三级。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而如果在官场上，25年前在厦门工作，今天你祖上就烧高香了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而如果一旦站错呢。则是身败名裂。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          都不说康熙大老虎了，这二年反腐抓了这么多部局级干部。每一个大佬的倒台，其实都意味着条线下面无数小弟的倒霉。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          四）流派的意义
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我们把流派分为二大类：资产类和人力类。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          其根本的意思，是讲二派赚钱的侧重点完全不同。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          无论他们的创意如何辉煌，如何精巧，归根到底，他们是靠自己的
          <span style="text-decoration:underline;">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<span style="font-size:16px;line-height:150%;font-family:楷体">

<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          人力类是动手，资产类是动脑。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          当你开始“博”房产升值时，其实你开始做的是“
          <span style="color:blue">
           投机
          </span>
          ”生意。连开了十五把大，第十六把是“大”还是“小”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          是飞黄腾达，还是倾家荡产。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          投机生意和卖劳动力的生意，完全完全不同。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          当然，在回报方面，差异也是巨大的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          人类回报最丰厚的是“担风险”的事情。
          <span style="color:blue">
           发财最快的，是你担了风险，并且押对了方向。
          </span>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果你仅仅是想每单赚50W，一年做三单。请改入人力类门下，不用再看下去了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          对于资产类的科技树，他最关心的，是“房价是涨还是跌”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          套利和套险”
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          精算”
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          头寸的建立”
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          细分结构”
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          人口趋势”
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          货币经济学”
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          可见，“资产类”炒房者的科技树，和“人力类”完全不一样。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我们用了约50章的篇幅，从《职业炒家》系列，写到《上海楼市的三分天下》系列，写到《炒房客的各大流派》系列。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          之所以在这里提到八大流派，描述框架却又不深入细说。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          是因为我们要表达的，这八大流派都是支路。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          科技树是长歪掉的，是长到分杈路上去了。宛如人类的进化拐成了三文鱼。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而我们之后的篇幅，是顺着“
          <span style="color:red">
           囤房待涨
          </span>
          ”这条路走上去的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          下面几乎所有的篇幅，都是沿着“囤房—赌大小”这条树干道讲下去的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          八大流派既然是岔路，所以就放在这里较前的位置。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          草草介绍，算交代了一个进化方向。以后分道扬镳了。
         </span>
</p>
