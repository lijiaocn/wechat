<h1 style="line-height:150%">
<span style="font-family:宋体;color:#984807">
          长线负债率
         </span>
<span style="color:#984807">
<sub>
           #2080
          </sub>
</span>
</h1>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在真正的财务模型中，“负债率”应该取22%左右。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          一）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          贷七成
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          人的一生中，
          <span style="color:red">
           大约只有一次机会
          </span>
          ，负债率可以到70%
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          看什么看，说的就是你。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这“一次机会”，说的就是你现在买房。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          譬如你买一套100W的房子，首付30W，贷款70W。负债率=“负债/总资产”=70%
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这个就是极限了。往前三十年，往后三十年，再也不可能达到。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          为了达到这个70%的“负债率”，你还需要一些额外条件。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          1）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你是按照“上限”贷款的。很多人买房时其实贷款完全不贷足。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          2）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你没有“其他”任何资产。而一般人总还有一点储蓄，股票，基金。此外契税和装修费其实也是“资产”。所以总负债率是到不了70%的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          3）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          仅限于“刚买房”
          <span style="color:blue">
           第一天
          </span>
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          关于第3点，让我们看一个展开说明。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          假如你五年前买了100W的房子，首付30W，贷款70W。那么仅有“交易中心”过户那一天，你是70%负债率的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          此后，房屋是“不断”在升值的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在通货膨胀的大环境下，任何物资价格都在上涨。80年代一个馒头5分钱，现在则要5元。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          五年之后，一般京沪100W的房价都涨到了300W。而你的贷款在不断归还。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          假设等额本息贷款还剩下66W。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          则“五年之后”你的负债率是66/300=
          <span style="color:red">
           22%
          </span>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          二）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          长线负债率
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          现在我们要问一个问题；
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “
         </span>
<span style="font-size:16px;line-height:150%;font-family:黑体;color:red">
          在
          <span style="text-decoration:underline;">
           专业人士
          </span>
          的眼里，楼市实际可以达到的杠杆率是多少
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          ”
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          关于这个问题，没有规范的答案。我只能给出一个基于经验和“实战”，切身经历大致实践的答案：
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:黑体;color:blue">
          长线负债率在22%左右。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这句话什么意思呢。让我们从另外一个角度来考虑问题。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          很多文科生小编一张嘴，“楼市充满了炒作和泡沫，炒房客都是90%的负债率”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          通常文科生的数学都不太好，智障也有问题。90%这种事俺早就不提了。坦白说，连50%都不提了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          50%
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这个目标，今生无望，下辈子再怎么努力也没有希望。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          为什么，因为
         </span>
<span style="font-size:16px;line-height:150%;font-family:黑体;color:blue">
          50%
         </span>
<span style="font-size:16px;line-height:150%;font-family:黑体;color:blue">
          负债率就意味着负债：净资产=1：1
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          假设你有5000万净资产吧。5000W净资产在多军内部不算太有钱的特例。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          则你的负债也要有5000W。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而负债5000W是什么概念呢。
          <span style="color:red">
           大概是月供50W
          </span>
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我敢打赌在座的没有一个经历过“月供50W”。你们也不可能开得出这样的收入证明。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          有5000W净资产的人很多，但他们一般都对应1100W多一点的“负债”额。月供在12W左右。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          同样道理，如果你的负债率是33%，则意味着净资产：负债=2：1
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          负债率25%，意味着净资产：负债=3：1
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          负债率20%，意味着净资产：负债=4：1
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          就连20%也是千难万难，万难百万难。
          <span style="text-decoration:underline;">
           绝大多数的多军负债率在9%以下
          </span>
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          三）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          负债率不变
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:blue">
          仅仅是保持着“负债率不变”，就是一件异常困难的事
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          以前我们举过一个“蚂蚁变大象”的例子。但是，其实各位只要想一想，假如你从A6789升级的过程中，始终“
          <span style="color:blue">
           负债率不变
          </span>
          ”，会发生什么情况。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          答案是：
         </span>
<span style="font-size:16px;line-height:150%;font-family:黑体;color:red">
          贷款必须和净资产一样速度增长
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而净资产的增长，是
          <strong>
           非常非常
          </strong>
          迅速的。在最初的几年，净资产很有可能是五倍，十倍的增长速度。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          相应的，
         </span>
<span style="font-size:16px;line-height:150%;font-family:黑体;color:red">
          贷款几乎没有增长的方法
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          当你躺在家里睡大觉，“内环线最美丽”一觉醒来，你的房产或许又升值了一百万元。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          可是贷款是需要你“一笔笔去办”的。需要你一笔笔去申请的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            攒几百万贷款，和攒几百万首付一样困难。
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            贷款赶上增长的速度，和打工赶上房价速度一样困难。
           </span>
</p>
</li>
</ul>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          闲话少说，如果你想“促使”你的负债增长，你基本上只有几种方法。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          1）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          新买房子，按揭贷款70%
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          2）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          抵押贷款
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          3）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          信用卡
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          4）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          私人借款
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          让我们看一下，“按揭贷款”的借贷率是70%，但是“按揭”严格地受“限贷”限制。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          基本上，每个身份证就只有一次机会。到了后期完全可以忽略。你的房子肯定远远比身份证多。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而“抵押贷款”理论上是可以贷七成。但在实际操作中，银行的普遍一种实践就是“估不足价钱”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          好比你一套1000W的房子，银行估价一般就是700W。然后在700W的基础上再借给你490W。除非你姓赵。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这事是如此地习以为常，泛滥到我们都已经完全习惯了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因此我们做预算的时候。通常“抵押”就是五成。预计五成，计划五成，接受五成。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在小学数学教育中，常常有一道幼儿题。你拿一杯“50%盐含量”的水去兑另一杯“10%盐含量”的水。请问盐浓度上限是多少。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          毫无疑问，上限就是50%。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          所以抵押贷款你申请得再多，无非“以盐兑水”。再怎么办也不能令你的负债率超过50%的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>

<p style="line-height:150%">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          对于文科生小编来说，肯定张口又吹“30年”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          对于许多普通人来说，大致心里有概念，约26
         </span>
<span style="font-size:16px;line-height:150%">
          ~
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          年的样子。因为还了几年。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而我们给出的答案是：“8年”。
         </span>

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">

         </span>

         </span>

<p style="line-height:150%">

<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          为什么，还是那个“盐水兑水”的问题。
         </span>

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “
         </span>

          按揭”贷款是非常偶然，特例，一辈子只有一次的事。“30年期，七折，七成”绝对是政府的福利。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          大写的二个字：
         </span>
<span style="font-size:35px;line-height:150%;font-family:黑体;color:blue">
          福利
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          根据“盐水兑水”的原理，你的“平均贷款年限”上限就不可能超过10年。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而另一方面，你又不可能“刚刚”办好抵押贷款，刚从银行大门出来。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          假如你手里有N套房子，并且成一个梯次运作的话。则你手中五套房子，剩余“贷款年限”很有可能是10，9，8，7，6年。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这样平均下来就是8年。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          五）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          财务远景图
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          综上，我们拼凑出了一幅“长期”的财务远景图。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          假设一个人是坚定的多军，他尽了一切力量来融资，
          <span style="color:red">
           来“活化”去角质化
          </span>
          ，则最终他能够达到的财务状态，也不过是：
          <span style="color:blue">
           负债22%，贷8年
          </span>
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          设想一下，假设一个人拥有5000W资产。则：
         </span>
</p>
<ul class="list-paddingleft-2" style="list-style-type: disc;">
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            净资产3900
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            负债1100
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            贷款8年，月供12W
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
            房租假设1.5%，每月6W
           </span>
</p>
</li>
<li>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:red">
            亏空6W
           </span>
</p>
</li>
</ul>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这个已经是极限了。负债率只能到22%，再也上不去了。这里面有二个原因，
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          1）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          贷不出
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          2）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          还不起
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          六）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          财务的分析
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “长期负债率”告诉我们几个道理。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          首先，文科生小编天天倡导的“房地产泡沫”“房贷泡沫”，是根本都不成立的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          只有实战过的人，才知道其中的辛苦。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          增加贷款”此事千辛万苦。严格地说可以认为是没有任何现成通道可以增加贷款。而让贷款和房价保持“同步增长”，更是一件不可能完成的事情。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          一个人一辈子只有一次机会，能够达到“负债率70%”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而这个“负债率70%”恰恰不是在多军手中完成的。相反是由“傻空”完成的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          只有“傻空”第一次买房，他们买房的
          <span style="color:blue">
           第一年
          </span>
          ，他们才是最脆弱的。最有可能负资产的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而对于多军，“负资产”这种事我早已绝望。甚至连负债率33%也已经绝望。负债率25%我都怀疑这辈子能否达成。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          一般认为，即使是“香港97”这种级别的大跌，最多回撤也就-30%。天塌下来有傻空顶着，政府会优先照顾烧炭的人的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          其次，“第一个1000W”最容易赚。第二个，第三个，就难赚得多。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为人一辈子只有一次机会“负债率70%”，在这个状态下，只要房价大涨一轮，分分钟资产爆倍。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但在后期，在老牌多军手里，杠杆负债率只有22%。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          第一个一千万最好赚。新人和资深炒家的相对距离拉近是很容易的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          第三，我们算一算多军“保持队伍”有多不容易。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          保持队伍”的定义，是随着社会财富的不断增长，维持着“阶层等级”不下降。社会精英位置不退后。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这个速度要多少呢。在《
          <a data_ue_src="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=206545656&amp;idx=1&amp;sn=3aefae067586f6dba726a16ddaaeeceb&amp;scene=21#wechat_redirect" href="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=206545656&amp;idx=1&amp;sn=3aefae067586f6dba726a16ddaaeeceb&amp;scene=21#wechat_redirect" target="_blank">
           向上爬 #2120
          </a>
          》一文中说是24%
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在卷2列篇中说是25
         </span>
<span style="font-size:16px;line-height:150%">
          ~
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          30%
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但是我们要注意另一件事，这个25
         </span>
<span style="font-size:16px;line-height:150%">
          ~
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          30%
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          的“财富增长率”，你是必须在
          <span style="color:blue">
           22%
          </span>
<span style="color:blue">
           的
           <strong>
            极低
           </strong>
           杠杆率
          </span>
          之下取得的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          许多人都有财富幻觉。他们刚结婚的几年，财富翻倍非常快。动辄50W变成500W。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但其实这是建立在“一生一次”70%负债杠杆率之下的。在“贷款七成”的情况下，楼价翻一倍，你就翻了400%
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但是，如果你的“长线杠杆率”只有22%，再要获得如此回报，就殊为不易。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          假设：
         </span>

<span style="font-size:16px;line-height:150%;font-family:楷体">
          可见，你所处的“城市”大势必须做到+20%的增长，这样你在微弱杠杆的情况下，才能勉强做到25%的复合增长。

<span style="font-size:16px;line-height:150%;font-family:楷体">
