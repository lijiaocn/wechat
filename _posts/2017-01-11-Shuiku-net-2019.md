<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          少年，我看你骨骼清奇，面相不凡。我这有本武功秘籍…………
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<span style="font-family: 楷体; font-size: 16px;">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          一）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          速成
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          水库《
          <a data_ue_src="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=2651015408&amp;idx=1&amp;sn=f4e471d469c4d50ab072d3879af2ccac&amp;chksm=80721ee3b70597f5a47bd472c5b70e96f9910613003e58ceaf0c57d001d5e11c7128887d0cda&amp;scene=21#wechat_redirect" href="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=2651015408&amp;idx=1&amp;sn=f4e471d469c4d50ab072d3879af2ccac&amp;chksm=80721ee3b70597f5a47bd472c5b70e96f9910613003e58ceaf0c57d001d5e11c7128887d0cda&amp;scene=21#wechat_redirect" target="_blank">
           速成篇
          </a>
          》这个标题一出来，顿时有人一拍脑袋。原来还有这好东西。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这事好呀。你说水库论坛从2015.2月，前前后后也写了二年多了吧。文章累积不下百万字。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          200
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          多篇文章，如果要看，得几天时间。看得累呢。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你说，如果可以浓缩到1000个字。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          您早说嘛。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          俺就不看正文了。只看速成版。一千个字岂不是三分钟就看完。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          为此，我们先要解释一下“速成”的意思。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          什么叫做“速成”。真正按照严格意义上讲，是没有“速成”这个说法的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          人类的社会，经过千淬百炼。真正能够保留下来的那个方法，一定是最有效率，最迅捷的方法。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          此所谓“正道”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而其他所有的“速成”方法，无一例外，都是有各种各样的缺陷的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          就好比你要学知识的话，老老实实地念完四年本科。这才是最踏实的。最踏实的才是最迅速的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而如果你要“一年，二年”就毕业。则多半是技校，大专。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我们今天说的水库“速成卷”。其实是
         </span>
<span style="font-size:16px;line-height:150%;font-family:黑体;color:red">
          通用件和专用件的区别。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          学问之道，分为“体、用、术”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
<img data-ratio="0.995253164556962" data-s="300,640" data-src="" data-type="png" data-w="632" src="{{ '/img/Ok4hZ0tV6r6Pq4s8kHZuenXtewWMcROdAUPpUYJJ4p7lYBZDqicGOjuHicNicbYibHCdMUdPPGuQxZ80EG88LVOB9A.png' | prepend: site.img | replace: '//','/' }}"/>
<br/>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “体”为根本。奥派和经济学的大道理，才是演化以后一切的基础。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而大道理并不能转换成“钱”。有时候你看“体”，洋洋洒洒写了几十万字，可是没一款能帮助你赚到钱。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          未免显得十分屈闷。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “速成篇”，则是侧重于“术”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          基本上世面上所有的“速成”，讲的都是“术”的道理。任何教你快速赚钱的方法，多半是一门
          <span style="color:#984807">
           手艺
          </span>
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          一个熟练的技工，三五天就可以上工了。而培养一个大学生，却需要四年。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “体”和“术”有什么区别呢。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          区别就在于“通用件”和“专用件”的区别。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          学习“术”的人，好比一具专业刨螺丝钉的机床。你让他加工螺帽，则效率极高，成本也节省。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          哪天你想加工一个曲轴，他就完全不能胜任了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我们的时代在变迁，社会在变迁。更要命的是政府也在变迁，环境和政策一天一个样。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这导致我们今天学的“
          <span style="color:blue">
           整套手法
          </span>
          ”，到明天可能就没用了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          好比第24，25条。我们说我们需要“二张房票”。其中A用于凤变冰，积累原始资金。B用于大面积低单价流派。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这就是11.28“新政”之后的新改变。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在之前，只要“首房无贷”，没有贷款记录也是可以认首套的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你只要A一缕房票就够了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          如果说这还属于“小改动”的话。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          则随着“水库”的普及面不断提高。目前市场上“
          <a data_ue_src="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=2651015136&amp;idx=1&amp;sn=6a26f814fc7006f64410109f707b7c11&amp;chksm=807211f3b70598e51c14a13b5658fc6d84295bfc95b47ef3b54b5f02eb65c3ed29d61e25a7ba&amp;scene=21#wechat_redirect" href="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=2651015136&amp;idx=1&amp;sn=6a26f814fc7006f64410109f707b7c11&amp;chksm=807211f3b70598e51c14a13b5658fc6d84295bfc95b47ef3b54b5f02eb65c3ed29d61e25a7ba&amp;scene=21#wechat_redirect" target="_blank">
           凤变冰
          </a>
          ”也越来越难找。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          凤变冰的特点，在于凤姐，不在于冰冰。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          凤姐才是你可以赚取的利润。而冰冰溢价少，最多使你卖得快一点。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          随着水库大军越来越多，凤姐都快被
          <span style="color:blue">
           挖光了
          </span>
          。洼地也是越来越少。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          整个逻辑链上，“凤变冰”也属于很重要的一环。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          失去了凤变冰，资本原始积累如何走。
          <a data_ue_src="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=2651015162&amp;idx=1&amp;sn=deb819ce8de024453f19f5b414847863&amp;chksm=807211e9b70598ffda556c62cbc3326e28a6fe734297c36168acc6c8b891aefe6b16551b3223&amp;scene=21#wechat_redirect" href="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=2651015162&amp;idx=1&amp;sn=deb819ce8de024453f19f5b414847863&amp;chksm=807211e9b70598ffda556c62cbc3326e28a6fe734297c36168acc6c8b891aefe6b16551b3223&amp;scene=21#wechat_redirect" target="_blank">
           如何获取家人的信任
          </a>
          ？
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这些东西，光靠一篇“速成篇”是无法回答以上问题的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          1000
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          个字可以速成。但速成的是“专用件”。只能用于
          <span style="text-decoration:underline;">
           特定时期特定场合特定人群。
          </span>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而只有把“体，用”都学会了。从头开始学，把基础打好。
          <span style="text-decoration:underline;">
           把整个逻辑讲透。
          </span>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          当外部环境变了，你可以自己
          <span style="text-decoration:underline;">
           顺着逻辑推导。依然有对策。
          </span>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这样100万字的阅读，才算是没有白费。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          二）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          小抄
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          速成篇的第二个问题。在于我们一直有一个梦想。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在《射雕英雄传》里面，铜尸陈玄风盗得半部九阴真经。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          他放在哪里都不放心，最后把原文毁了。刻成刺青，缝在自己的胸口。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          后来被人皮剥下。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          八卦读者看到这里，不由得会问。“在古代又没有打印机，您最小用几号字体”？
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          人的胸口皮肤，面积十分有限。在这么小的范围内，你怎样抄下半部《九阴真经》。能写多少字。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">

<p style="line-height:150%">
<span style="font-family: 楷体; font-size: 16px;">

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          字数不要多，就1000个字左右。足够刺绣刻在陈玄风的胸口。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          足够做成小抄，在大学生之间流传。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          所以才有了#2760这个“速成版”版本。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

           经济学原理
          </a>
          dT&gt;0的，一概不要不录取。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          最终勉勉强强凑了1009个字。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
<br/>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          但其实还是很偏向“用”的。介于“用、术”之间。比较偏向
          <span style="text-decoration:underline;">
           蛮理论化的。
          </span>
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为，真正要写“速成版”的话。
         </span>
<span style="font-size:27px;line-height:150%;font-family:楷体">

          我只要43个字就够了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<blockquote>
<p style="line-height:150%">
<span style="font-size:19px;line-height:150%;font-family:楷体">
           “
          </span>
<span style="font-size:19px;line-height:150%;font-family:黑体;color:red">
           七成30年按揭，去买上海康城一楼带地下室的户型。按照2017年1月的价格，闭着眼睛买。即买即赚
          </span>
<span style="font-size:19px;line-height:150%;font-family:楷体">
           ”！

</blockquote>
<p style="line-height:150%">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          三）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          工作
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          好了，现在我们具体展开，“速成篇”66条究竟都写了些什么东西。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          首先，你需要一份工作。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          工作，绝对不是让你去赚工资的。更不是让你去拼搏升职加薪。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          事实上，既然是“速成版”。则时间是非常宝贵的。一般在36个月内就要结束战斗。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          36
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          个月，靠工资，那是根本攒不了多少钱。升职提干也就是小组长，毫无意义。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “上班”的主要用途，是为你
         </span>
<span style="font-size:21px;line-height:150%;font-family:楷体;color:blue">
          交四金
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体;color:blue">
          。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          交四金有非常重要的意义。尤其是当申请房贷，信用卡时，会有很大的加成。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你只要有一份体面正当的工作就可以了。工资无所谓。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “上班”的第二个含义。是结交一群“小姐妹”。或者“小兄弟”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体;color:red">
          赚钱再快，也比不上申卡。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          对于一个初出茅庐，刚刚踏上社会的年轻人。如果他要靠“工资---积蓄”赞起人生的第一个150W，那可能需要五六年，甚至更长的时间。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而办卡的话，体制内人均130W，体制外人均80W。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          也就是你那群小姐妹，一个个都是人形金矿。只要你挖掘得好，四五个朋友一凑就是300W启动资金。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          四）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          流水
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          一般银行的需求，让你打六个月的流水。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          上海最近四季度管得十分紧，甚至要求你打十二个月的流水。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          无论如何，你在公司里得待六个月。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在这段时间内，你也不会闲着。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          一件可以做起来的事，是“养流水”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          你要严格地做到“收支二笔账”，甚至可以说，同一家工行开2张借记卡。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          其中，所有的“支出”走一张卡。譬如月供，还信用卡，分期付款。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          所有的“收入”走一张卡。工资，房租，定期收入。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这样，等信贷主任问你要“流水”时。“收入卡”才会好看。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而对于“初入职场”的年轻人来说，工资不会太高。“流水”也十分难看。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          这时候，你还可以“养流水”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          也就是说，每个月固定有一笔23456.78元的进账。放在卡上，暂时不要取走。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          每三个月可以集中取走一笔60000.
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<blockquote>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
           按照谨慎的原则，银行应该集中监督你“工资”项目下的每月定期收入。
          </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
           但我们实战体会，只要你每月固定日期，非“工资”类目其实也很有用。
          </span>

</blockquote>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          则等到你去打存折簿时，他显示也是“工资”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          具体不展开，免得洞塌。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          总而言之，你的流水得养起来。等哪天真用上贷款时，谋而备就可以用。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          除了流水之外，另一个可以养的是“社保”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

         </span>

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>

<span style="font-size:16px;line-height:150%;font-family:楷体">
          当你看见职业炒家唰唰唰地“万箭齐发”时。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          其实人家二年前就开始准备了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          五）
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          扫街
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在这六个月内，除了养流水，另一个可以做、必须做的事情是“扫街”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          天底下本没有捷径，也没有速成法。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          一切都是无差别的人类体力劳动。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">

<span style="font-size:16px;line-height:150%;font-family:楷体">
          也就是只要一年的时间，你就可以；
         </span>
</p>

            脱离赤贫

</p>

<p style="line-height:150%">

<p style="line-height:150%">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          在水库上，常有“一年时间10000到1000万”的故事流传。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          其实前半截都在建仓。建仓后楼市一
         </span>
<span style="font-size:24px;line-height:150%;font-family:楷体">
          冲
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          就
         </span>
<span style="font-size:24px;line-height:150%;font-family:楷体;color:red">
          千万
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          了。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          十三）
          <span style="font-variant-numeric: normal;font-stretch: normal;font-size: 9px;line-height: normal">
</span>
</span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          2N
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          中级的后续做法是2N，也就是每年买二套，持续扩大。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          顺着这条线，最终可以一直做到A9.
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          至于
          <a data_ue_src="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=2651015141&amp;idx=1&amp;sn=63f17628cbcf2fbfdafac93d4f077392&amp;chksm=807211f6b70598e0716fd897fc53c1de95e7c9d8dfe58d9bf5feb94d0555ebb8e8532613bc08&amp;scene=21#wechat_redirect" href="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=2651015141&amp;idx=1&amp;sn=63f17628cbcf2fbfdafac93d4f077392&amp;chksm=807211f6b70598e0716fd897fc53c1de95e7c9d8dfe58d9bf5feb94d0555ebb8e8532613bc08&amp;scene=21#wechat_redirect" target="_blank">
           2N
          </a>
          的具体做法，水库文献中已经有了详细描述。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          不再赘述。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          值得一提的是，除了
          <a data_ue_src="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=2651014908&amp;idx=1&amp;sn=7fcf1537987d7e7fd5df0ac296ca0240&amp;scene=21#wechat_redirect" href="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=2651014908&amp;idx=1&amp;sn=7fcf1537987d7e7fd5df0ac296ca0240&amp;scene=21#wechat_redirect" target="_blank">
           抵押
          </a>
          ，
          <a data_ue_src="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=2651014912&amp;idx=1&amp;sn=42f1eacd72da0e7e87280d55f206937f&amp;scene=21#wechat_redirect" href="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=2651014912&amp;idx=1&amp;sn=42f1eacd72da0e7e87280d55f206937f&amp;scene=21#wechat_redirect" target="_blank">
           截断
          </a>
          ，
          <a data_ue_src="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=2651014886&amp;idx=1&amp;sn=fc7eb2dec44a760322d095087a44190e&amp;scene=21#wechat_redirect" href="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=2651014886&amp;idx=1&amp;sn=fc7eb2dec44a760322d095087a44190e&amp;scene=21#wechat_redirect" target="_blank">
           高评
          </a>
          ，
          <a data_ue_src="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=2651014922&amp;idx=1&amp;sn=1c214c174849286499f23a2e8efecbc0&amp;scene=21#wechat_redirect" href="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=2651014922&amp;idx=1&amp;sn=1c214c174849286499f23a2e8efecbc0&amp;scene=21#wechat_redirect" target="_blank">
           对敲
          </a>
          等各项手法之外。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          还有一个极重要的选项：“卖出”。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          因为楼市这玩意，也和桃子一样，存在一个“
          <a data_ue_src="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=401338731&amp;idx=1&amp;sn=afb1238dc2b229f0ccfc71915a07d039&amp;scene=21#wechat_redirect" href="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=401338731&amp;idx=1&amp;sn=afb1238dc2b229f0ccfc71915a07d039&amp;scene=21#wechat_redirect" target="_blank">
           成熟
          </a>
          ”的概念。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          一个地段从潜力，青涩，成长，发展；
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          最终会达到：成熟，过成熟，这二个阶段。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          其中“成熟”阶段，该板块已达最最热炒顶峰。继续成长可能性不大。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “过成熟”阶段，该板块已炒完审美疲劳。此后只会走下坡路。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          当手中物业接近“成熟”时，要尽早安排出售。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而物业“过成熟”时，要坚决全力抛出换筹。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          譬如说，97年的田林，03年的徐家汇，04年的莘庄，09年的联洋，就属于热到巅峰而衰弱的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          2016
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          年的大宁，唐镇，也处于“成熟水蜜桃”即将腐烂的状态。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          当投资者盘算好“
          <a data_ue_src="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=401353641&amp;idx=1&amp;sn=81e4afbf77a73e726f0a24f624d1c980&amp;scene=21#wechat_redirect" href="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=401353641&amp;idx=1&amp;sn=81e4afbf77a73e726f0a24f624d1c980&amp;scene=21#wechat_redirect" target="_blank">
           出售计划
          </a>
          ”，一心一意想抛时。突然遇见政府宏观调控。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          那定会是非常愤怒的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          十四）
          <span style="font-variant-numeric: normal;font-stretch: normal;font-size: 9px;line-height: normal">
</span>
</span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          高级阶段
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          高级阶段，主要是守。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          “一切附加值都不保值”，既然如此，上海二个字也是不保值的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          我宁愿逐步的按照10：1比例，把资产转移到重庆去。
         </span>

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

</span>

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">

<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          十五）
          <span style="font-variant-numeric: normal;font-stretch: normal;font-size: 9px;line-height: normal">
</span>
</span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          终极阶段
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          对于终极阶段来说，富贵浮云，都是白云苍狗。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          不仅“上海”二个字不保值，就连“财富”二字，本身也是
          <a data_ue_src="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=207860847&amp;idx=1&amp;sn=c85f14f9679575b05bd0afefd9f5665c&amp;scene=21#wechat_redirect" href="http://mp.weixin.qq.com/s?__biz=MzAxNTMxMTc0MA==&amp;mid=207860847&amp;idx=1&amp;sn=c85f14f9679575b05bd0afefd9f5665c&amp;scene=21#wechat_redirect" target="_blank">
           不保值
          </a>
          的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          财富并不能抵抗战争，政治，社会动荡。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          甚至科技发展……
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          象王健林有5000亿财富，他的下一代，下二代，下三代，再要维持他在全社会的财富水准，那是非常非常难。几乎注定要降阶的。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          相对而言，如果王健林有5000个儿子，而每个儿子均分1亿。并且散居在全国150个万达广场，全国各地。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          则“王氏家族”堪称是根深蒂固，几乎无可撼动。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          5000
         </span>
<span style="font-size:16px;line-height:150%;font-family:楷体">
          人的财富总量增长，也一定会超过王思聪一个人的速度。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          孩子才是终极财富。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
</span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          血脉的传承，就是自然界的
          <span style="color:red">
           究极
          </span>
          意义。
         </span>
</p>
<p style="line-height:150%">
<span style="font-size:16px;line-height:150%;font-family:楷体">
          而你要做的，则是让尽可能多的孙辈来到这个世界上。
